1. آلية عمل أداة Comparer وأهميتها الميدانية

تُعد أداة Burp Comparer العين الميكروسكوبية لباحث الـ Bug Bounty. تتيح لك الأداة إجراء مقارنة مرئية سطر بسطر وحرف بحرف بين نصين أو استجابتين HTTP للتقاط الاختلافات الدقيقة التي قد لا تلاحظها بالعين المجردة.

📊 الفرق بين مقارنة الكلمات ومقارنة البايتات:

  • Words Comparison: تقارن الكلمات والجمل في النصوص وتظلل الكلمات التي تم تعديلها أو إضافتها أو حذفها باللون الأخضر والأحمر.
  • Bytes Comparison: تقارن النصوص على مستوى القيم السداسية عشرية (Hex/Binary Bytes)، وهي ممتازة جداً لاكتشاف اختلاف الحروف غير المرئية أو الـ Null Bytes.
💡 متى تستخدم Comparer في الصيد الحقيقي؟
• عند مقارنة استجابة تسجيل دخول ناجح مقابل فاشل لاكتشاف الـ Oracle Leaks.
• عند اختبار ثغرات الـ Blind SQL Injection لمعرفة هل يتغير حجم الـ Response ببايت واحد عند تحقق الشرط الصحيح (`1=1`).

🧪 مختبر 1: محاكي مقارنة صلاحيات الحسابات (Authentication Diff Sandbox)

اضغط على زر تشغيل المقارنة لرؤية أداة Comparer وهي تظلل الفروق الحساسة بين استجابة حساب مستخدم عالي الصلاحيات (Admin) ومستخدم عادي (User):

[ITEM A: Normal User Response]
HTTP/1.1 200 OK Content-Type: application/json {"status": "success", "role": "user", "access_level": 1, "is_vip": false}
[ITEM B: Admin Response]
HTTP/1.1 200 OK Content-Type: application/json {"status": "success", "role": "admin", "access_level": 9, "is_vip": true}
