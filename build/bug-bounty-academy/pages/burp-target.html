⭐ Target & Scope Masterclass

الدليل الشامل لإدارة شجرة هيكل الموقع، ضبط نطاق الاستهداف بـ Regex، واكتشاف الأصول المخفية.

1. هيكل شجرة الموقع (Site Map Architecture) وإدارة النطاق (Scope)

تُعتبر لوحة Target في Burp Suite المركز الرئيسي لرسم خريطة الهجوم الميدانية. أثناء تصفحك للموقع المستهدف، تقوم الأداة بتجميع وتوليد شجرة شجرية تفاعلية (Hierarchical Tree) تضم كل المجالات، المجلدات، والملفات التي تم اكتشافها.

🎯 لماذا يُعد ضبط النطاق (Target Scope) أمر حاسم؟

  • تجنب الحظر الميداني: يضمن عدم إرسال هجمات أو استعلامات إلى سيرفرات خارجية غير مصرح بها (مثل استضافات الـ CDN أو تحليلات Google).
  • تصفية الضوضاء: يتيح لك تصفية الـ HTTP History والأدوات لتظهر فقط الأصول المستهدفة في برنامج الـ Bug Bounty.
  • تسريع عملية الزحف: حصر محرك الـ Crawler في المسارات المصرح بها فقط.
💡 نصيحة احترافية في الـ Regex: عند إضافة نطاق إلى الـ Target Scope، استخدم تعبير الـ Regex التالي ليشمل كل Subdomains الهدف:
^.*\.target\.com$

2. أدوات المشاركة واكتشاف المحتوى المخفي (Engagement Tools)

بضغط زر يمين على أي مجلد داخل الـ Site Map، تتيح لك Burp Suite أدوات متقدمة لاكتشاف الأسطح الهجومية:

  • Content Discovery: أداة سريعة تقوم بالتخمين الهيدروليكي على الملفات والمجلدات المخفية (مثل `.env`, `backup.sql`, `config.php`).
  • Find Comments / Scripts: استخراج كل التعليقات البرمجية وملفات الـ JavaScript المكتشفة في الموقع لمراجعة الـ API Keys المخفية.
  • Analyze Target: تحليل شامل يعرض لك عدد المعاملات (Parameters) والـ MIME Types المصنفة في الموقع.

🧪 مختبر 1: مستكشف شجرة الموقع (Site Map Tree Sandbox)

قم بالضغط على المجلدات في شجرة الموقع المحاكات التالية لاكتشاف المسارات الحساسة والملفات المخفية:

target-vulnerable.com
/api/v1
/users (GET)
/auth (POST)
/internal_admin [RESTRICTED]
db_backup.sql

🧪 مختبر 2: ضابط نطاق الاستهداف (Target Scope Regex Engine)

قم بتجربة كتابة تعبير Regex لاستهداف `company.com` وكل فروعها، واضغط فحص لمعرفة هل الطلب داخل النطاق أم لا:

Challenge Mode (Target CTF)

يوجد مسار حساس مخفي داخل شجرة الموقع يحتوي على مفتاح تسريب سري. استكشف المسارات واكتشف الـ Flag!
