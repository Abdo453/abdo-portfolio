Interactive Exploitation Cheat Sheets Hub

المركز المرجعي التفاعلي الشامل لأوراق الغش وشجرات القرارات وشفرات التخريش لجميع ثغرات الـ Web Security الفائقة.

1. SQL Injection Decision Tree

مرجع كامل لدمج النصوص والتأخير الوقتي واستخراج البيانات عبر أخطاء النظام و Burp Collaborator DNS لكافة قواعد البيانات (Oracle, MSSQL, Postgres, MySQL).

افتح مرجع SQLi

2. XSS Context Payload Matrix

مرجع سياقات الـ XSS وتجاوز الـ WAF والـ Filter في سياقات HTML, Attribute, JavaScript, و DOM-based vulnerabilities.

افتح مرجع XSS

3. SSRF Decision Tree

شجرة قرارات استغلال الـ SSRF للوصول لسيرفرات الـ Localhost وتجاوز الفلاتر وسرقة مفاتيح الكلاود (AWS Metadata 169.254.169.254).

افتح مرجع SSRF

4. JWT Attack Matrix

مرجع اختراق توكنات الـ JWT عبر `alg: none` وتزوير المفاتيح العامة (JKU / JWK Injection) والكسر بالـ Brute-force.

افتح مرجع JWT
