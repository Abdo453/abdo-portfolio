1. Introduction & Root Cause

تعريف الثغرة: حقن أوامر SQL هو ثغرة تسمح للمهاجم بالتلاعب بالاستعلامات المرسلة لقاعدة البيانات وعرض بيانات غير مصرح بها.

السبب الجذري: دمج مدخلات المستخدم مباشرة في استعلامات SQL بدون استخدام Prepared Statements أو Parameterized Queries.

2. Testing Checklist (GET, POST, JSON, Cookies)

Test parameters with Single Quote (') and Double Quote (")
Check JSON payloads in POST request body
Verify Cookie headers and Custom Session parameters
Test for Time-based delays in hidden headers

3. Attack Flow & Burp Workflow

Proxy Intercept ↓ Repeater (Confirm Bug) ↓ Intruder (Brute-force columns) ↓ sqlmap (Exfiltrate DB)

4. Cross-Chaining & Escalation

SQL Injection ↓ Read Database Hashes ↓ Hash Cracking / Admin Access ↓ File Upload Admin Shell ↓ Remote Code Execution (RCE)

5. Top Exploitation Payloads

' OR 1=1--
' UNION SELECT NULL, username, password FROM users--

6. Real Bug Bounty Story

الهدف: شركة تقنية كبرى (برنامج HackerOne).

القصة: اكتشف باحث بارامتر مخفي في خاصية الفلترة، وقام بحقن استعلام UNION لاستخراج جدول الـ Administrators بالكامل، وحصل على مكافأة قدرها $5,000.

1. Introduction & Root Cause

تعريف الثغرة: تنفيذ الأكواد عبر المواقع يسمح بحقن سكريبتات خبيثة في متصفح الزوار لسرقة البيانات أو السيطرة على الجلسات.

السبب الجذري: عرض مدخلات المستخدم على المتصفح مباشرة دون عمل HTML Entity Encoding.

2. Testing Checklist

Test payload in HTML context
Test inside input tags & attributes
Check reflection in JavaScript variables

3. Attack Flow & Burp Workflow

Proxy Intercept ↓ Repeater (Confirm Reflection) ↓ DOM Invader (Analyze source/sink) ↓ Trigger Alert()

4. Cross-Chaining & Escalation

Stored XSS ↓ Session Hijacking (Cookie steal) ↓ CSRF Execution (Change email) ↓ Full Account Takeover

5. Top Exploitation Payloads

&lt;script&gt;alert(1)&lt;/script&gt;
&lt;img src=x onerror=alert(1)&gt;

6. Real Bug Bounty Story

الهدف: موقع تسوق شهير (برنامج Bugcrowd).

القصة: تمكن الباحث من حقن Payload خبيث داخل حقل عنوان الشحن، ليتم تنفيذه تلقائياً في لوحة تحكم الموظف بمجرد استعراض الطلب، وحصل على مكافأة بقيمة $3,000.

1. Introduction & Root Cause

تعريف الثغرة: تزوير الطلبات من جهة الخادم يسمح للمهاجم بإجبار السيرفر على إرسال طلبات لشبكات داخلية أو للـ Metadata الخاصة بالكلاود.

السبب الجذري: السماح للمستخدم بالتحكم الكامل في عنوان الـ URL الذي سيقوم الخادم بسحب البيانات منه دون التحقق من نطاقه.

2. Testing Checklist

Test with local addresses (localhost, 127.0.0.1)
Test cloud metadata endpoints
Use Collaborator to test Out-Of-Band SSRF

3. Attack Flow & Burp Workflow

Proxy Intercept ↓ Repeater (Inject Localhost IP) ↓ Collaborator (Validate connection) ↓ Metadata Exfiltration

4. Cross-Chaining & Escalation

SSRF ↓ Cloud Metadata (AWS IAM Credentials) ↓ AWS Admin Console access ↓ Remote Code Execution (RCE)

5. Top Exploitation Payloads

http://169.254.169.254/latest/meta-data/
http://localhost:80/admin

6. Real Bug Bounty Story

الهدف: منصة دفع شهيرة.

القصة: استغل الباحث ثغرة SSRF في خاصية تحميل الفواتير عبر رابط URL، ونجح في استخراج الـ Security Credentials الخاصة بسيرفرات AWS وحصل على مكافأة بلغت $8,500.
