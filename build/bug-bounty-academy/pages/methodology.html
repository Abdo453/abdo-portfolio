0. XSS Knowledge Map

1. Introduction & Contexts

HTML Context: يحدث عندما ينعكس مدخل المستخدم مباشرة بين وسوم الـ HTML مثل: <h1>USER_INPUT</h1>.

Attribute Context: يحدث عندما ينعكس داخل خاصية مثل: <input value="USER_INPUT"> (يتطلب الخروج من الخاصية باستخدام الكوتس ").

2. Testing Checklist

Test payload in HTML context
Test inside input tags & attributes
Verify DOM sinks like innerHTML and eval()

3. Attack Flow & Burp Workflow

Proxy Intercept ➔ Repeater (Confirm Reflection) ➔ DOM Invader ➔ Trigger alert/console.log

4. Secure Coding Mode

// [Vulnerable Code] let userQuery = getQueryParam("q"); document.getElementById("output").innerHTML = userQuery; // Vulnerable to DOM XSS
