بناءً على محتوى موقع NetworkLessons، إليك خطة متكاملة مقسمة إلى 8 مراحل رئيسية:

المرحلة الأولى: أساسيات الشبكات (أسبوعين)

الأهداف:

  • فهم مفهوم الشبكات وأنواعها (LAN, WAN, MAN)
  • نماذج OSI و TCP/IP
  • أنواع الكابلات (Copper, Fiber, Coaxial)
  • المفاهيم الأساسية: Bandwidth, Throughput, Latency

المواضيع:

  • Network topologies
  • Physical vs Logical topologies
  • Ethernet basics (CSMA/CD)
  • MAC addresses and framing

المرحلة الثانية: بروتوكولات الشبكات (أسبوعين)

الأهداف:

  • إتقان IPv4 و IPv6
  • فهم TCP vs UDP
  • نموذج Client-Server و Peer-to-Peer

المواضيع:

  • IPv4: Addressing, Subnetting, CIDR, VLSM
  • IPv6: Address types, Compression, Transition mechanisms
  • TCP/UDP: Ports, Three-way handshake, Flow control
  • Application Layer: DNS, DHCP, HTTP/HTTPS, FTP, SSH, Telnet

💡 نصيحة: تدرب يومياً على Subnetting لمدة 30 دقيقة حتى تتقنه

المرحلة الثالثة: أجهزة Cisco - الطبقة الثانية (أسبوعين)

  • Cisco IOS basics (CLI commands)
  • Switching fundamentals
  • VLANs: Creation, assignment, Inter-VLAN routing
  • Trunking: 802.1Q, Native VLAN, DTP
  • STP: PVST+, Rapid PVST+
  • EtherChannel: PAgP, LACP
  • Security: Port Security, DHCP Snooping, Dynamic ARP Inspection

المرحلة الرابعة: Routing - الطبقة الثالثة (3 أسابيع)

  • Router fundamentals and IOS
  • Static routing (IPv4/IPv6)
  • Default routes
  • Dynamic Routing: RIP, OSPFv2, OSPFv3, EIGRP
  • Routing table interpretation
  • Inter-VLAN routing (Router-on-a-stick, SVI)

المرحلة الخامسة: خدمات الشبكة (أسبوع)

  • DHCP: Server, Relay, Client
  • NAT: Static, Dynamic, PAT (Overload)
  • ACLs: Standard, Extended, Named, IPv6 ACLs
  • NTP, SNMP, Syslog, CDP, LLDP

المرحلة السادسة: أمن الشبكات (أسبوع)

  • Security fundamentals & Password encryption
  • SSH configuration
  • VPNs: Site-to-site, Remote access
  • Wireless Security: WPA2, WPA3
  • Port Security & AAA

المرحلة السابعة: الشبكات اللاسلكية (أسبوع)

  • Wireless standards (802.11a/b/g/n/ac/ax)
  • SSID, BSSID, ESSID
  • Wireless topologies & WLC basics
  • CAPWAP protocol & Roaming concepts

المرحلة الثامنة: Network Automation & Programmability (أسبوع)

  • SDN & Controller-based networking
  • REST APIs, JSON, XML, YAML
  • Automation tools: Ansible, Puppet, Chef
  • Python for network automation & NETCONF/RESTCONF
