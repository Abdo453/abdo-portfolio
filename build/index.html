What is John the Ripper? / ما هو؟

أداة سريعة للغاية لاختبار أمن وقوة كلمات المرور وفك وتخمين التشفير (Hash Cracking).

How it Works / كيف تعمل؟

تدمج بين هجمات القاموس (Dictionary attacks) والتحليل الذكي لإنشاء كلمات مرور مخصصة وقادرة على فك مئات الأنواع من الهاشات والملفات المضغوطة المشفرة.

Useful Commands / أوامر مفيدة

انقر على زر النسخ لنسخ الأمر مباشرة إلى الحافظة:

1. Crack Shadow Hashes / فك كلمات مرور النظام باستخدام قائمة كلمات: 
john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt
2. Show Cracked Passwords / إظهار كلمات المرور المخمنة مسبقاً: 
john --show hashes.txt
3. Specifying Format / تحديد نوع التشفير يدوياً لفك الهاش: 
john --format=raw-md5 --wordlist=rockyou.txt hashes.txt
4. Crack Zip Password / تخمين وفك تشفير ملف مضغوط ZIP: 
zip2john backup.zip > zip.hash && john --wordlist=rockyou.txt zip.hash
5. Crack PDF Password / تخمين وفك كلمة مرور ملف PDF محمي: 
pdf2john document.pdf > pdf.hash && john --wordlist=rockyou.txt pdf.hash
