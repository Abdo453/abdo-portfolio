Theory / النظرية

AR

المستوى 1: تسريبات مخازن S3 المفتوحة (S3 Buckets). يترك المطورون أحياناً صلاحيات قراءة مخازن التخزين مفتوحة للكل (Anonymous Access). استخدم أدوات المحاكاة مثل aws s3 ls لفحص المخازن مثل s3://hunteros-backups واستخراج الملفات السرية.

EN

Level 1: S3 Bucket Misconfiguration. Storage buckets left with public read access allow anonymous enumeration. Use aws s3 ls s3://[bucket] commands to audit storage nodes and retrieve backups.

AR

المستوى 2: الهروب من حاويات دوجر (Docker Container Escape). عندما تعمل حاوية Docker بصلاحيات مرتفعة (Privileged Mode)، يمكنها الوصول للقرص الصلب الفعلي للسيرفر المضيف. ابحث عن الأقراص الصلبة باستخدام fdisk -l أو افحص المجلدات، ثم قم بدمج/ربط (mount) القرص الفعلي /dev/xvda1 بمجلد داخل الحاوية لتجاوز العزل.

EN

Level 2: Container Escape. Privileged Docker containers can interact with host hardware. Use fdisk -l or lsblk to identify the host drive (e.g. /dev/sda1 or /dev/xvda1) and mount it to get root control of the parent host operating system.
