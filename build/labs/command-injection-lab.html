Source Code Remediation

Here you can view the vulnerable PHP code that parses your input behind the scenes, and how to secure it against OS Command Injection. 

<?php
if (isset($_POST['ip'])) {
    $ip = $_POST['ip'];
    // Vulnerable: Directly concatenating input!
    $output = shell_exec("ping -c 4 " . $ip);
    echo $output;
}
?>
<?php
if (isset($_POST['ip'])) {
    $ip = $_POST['ip'];
    // 1. Escape the shell argument safely
    $safe_ip = escapeshellarg($ip);
    // 2. Validate IP format (Strict constraint)
    if (filter_var($ip, FILTER_VALIDATE_IP)) {
        $output = shell_exec("ping -c 4 " . $safe_ip);
        echo $output;
    } else {
        echo "Error: Invalid IP Address format!";
    }
}
?>
