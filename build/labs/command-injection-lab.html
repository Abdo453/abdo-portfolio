مختبر حقن الأوامر التفاعلي

Interactive Command Injection Lab — 3 Levels

CRITICAL OWASP A03
AR

محاكي أداة تشخيص الشبكة (Ping Utility) الضعيفة. تلاعب بمدخلات الـ IP لحقن أوامر النظام وقراءة الـ Flags.

EN

Simulated vulnerable Network Ping utility. Manipulate the IP parameter to inject system commands and extract flags.

Level: 1/3 Flags: 0/3 Progress: 0%
ping -c 3
[*] Command Injection Lab initialized.
Enter an IP Address above to test network connectivity.
[*] Hint L1: Try injecting commands using separation characters like semicolon (;) or vertical bar (|). E.g.: 8.8.8.8; whoami

Theory / النظرية

AR

المستوى 1: حقن كلاسيكي. النتيجة تنعكس مباشرة في الاستجابة. استخدم ; whoami أو ; cat flag.txt.

EN

Level 1: Classic command injection. Output is returned in the response. Use ; whoami or ; cat flag.txt.

AR

المستوى 2: حقن أعمى (Blind). السيرفر لا يرجع مخرجات الأمر المحقون بل يظهر فقط نجاح/فشل الـ ping. استخدم الـ Time Delay مثل ; sleep 5 للاستكشاف.

EN

Level 2: Blind command injection. No command output is returned. Infer command execution using time delays like ; sleep 5.

AR

المستوى 3: جدار الحماية (WAF) يمنع الفراغات (Spaces). تجنب استخدام المسافات لتخطي الفلتر، وجرب بدائل مثل $IFS.

EN

Level 3: WAF blocks spaces. Bypass space filters using shell variables like $IFS.
