ROADMAP ⏱ 5 Stages

🎯 Roadmap Concept / فكرة المنهجية

AR

منهجية متكاملة مصممة لنقلك من الصفر في استخدام نظام لينكس إلى الاحتراف الكامل في إدارة الأنظمة، الأتمتة باستخدام Ansible، التحليل الجنائي الرقمي (Forensics) لاستخراج الأدلة بعد الاختراق، وفهم النواة (Kernel Internals) لتحليل الأداء والتطوير البرمجي. تعتمد الخطة على دمج منهج Red Hat الأكاديمي مع الكتب والمراجع القياسية عالمياً.

EN

A comprehensive study plan designed to transition you from Linux basics to systems administration, enterprise automation using Ansible, incident response / digital forensics, and deep kernel internals for performance analysis. This roadmap merges Red Hat academic training paths with industry-standard reference books.

▶ المرحلة 1: أساسيات النظام (RH124)
مستوى مبتدئ

الهدف: التعامل مع لينكس كمستخدم محترف وفهم البنية التحتية الأساسية لنظام التشغيل.

الموضوع المحتوى والتطبيقات المصدر المنهجي
مقدمة في لينكس توزيعات لينكس، نظام الملفات، الفرق بين Kernel و Shell RH124 Ch.1
سطر الأوامر (CLI) أوامر التنقل وإدارة الملفات ls, cd, cp, mv, rm, find, grep RH124 Ch.2
المستخدمين والمجموعات إضافة المستخدمين، الصلاحيات chmod, chown، آلية sudo RH124 Ch.4-5
العمليات (Processes) إدارة ومراقبة العمليات ps, top, kill، أساسيات systemd RH124 Ch.7
الشبكات والسجلات إدارة كروت الشبكة nmcli, ip، تحليل السجلات journalctl RH124 Ch.10-12

📖 كتب مقترحة للمرحلة: Linux Administration: A Beginner's Guide - Wale Soyinka

▶ المرحلة 2: الإدارة المتوسطة (RH134)
مستوى متوسط

الهدف: تأمين السيرفر، جدولة المهام، التخزين المتقدم واستكشاف أخطاء الإقلاع.

الموضوع المحتوى والتطبيقات المصدر المنهجي
التخزين والـ LVM التقسيم المنطقي LVM، التخزين الإضافي، تهيئة أنظمة الملفات RH134 Ch.4-5
SELinux للأمان إدارة صلاحيات SELinux، حل مشاكل المحتوى والسياقات (Troubleshooting) RH134 Ch.8
جدولة المهام والأمان جدولة cron والـ Systemd Timers، جدار الحماية firewalld RH134 Ch.9-10
التشغيل الآلي والإقلاع كتابة سكريبتات Bash، حل مشكلات الإقلاع (Rescue & Emergency Mode) RH134 Ch.11-13

📖 كتب مقترحة للمرحلة: UNIX and Linux System Administration Handbook - Evi Nemeth

▶ المرحلة 3: الأتمتة والاحتراف (RH294/RHCE)
مستوى متقدم

الهدف: إدارة خوادم متعددة في نفس الوقت، الأتمتة الصناعية (DevOps Basics)، والتعامل مع الحاويات.

الموضوع المحتوى والتطبيقات المصدر المنهجي
أساسيات Ansible تثبيت Ansible، إدارة الـ Inventory، فهم الـ Ad-hoc Commands RH294 Ch.1-2
Playbooks & Variables كتابة ملفات التشغيل الآلي YAML، إدارة المتغيرات، التحكم في التدفق RH294 Ch.3-5
الـ Roles والمهام المتقدمة بناء هيكل منظم للسكريبتات، التعامل مع الـ Templates والملفات الحساسة RH294 Ch.7-9
Containers & Podman إدارة الحاويات بدون Root باستخدام Podman على سيرفرات RHEL RHCE Syllabus

📖 كتب مقترحة للمرحلة: Ansible for DevOps - Jeff Geerling

▶ المرحلة 4: التحليل الرقمي الجنائي (Forensics)
مستوى خبير

الهدف: تحليل السيرفر بعد تعرضه للاختراق، فهم الأدلة الرقمية، واسترجاع الملفات وإعادة بناء الأحداث.

الموضوع المحتوى والتقنيات الأدوات المستخدمة
تحليل نظام الملفات تتبع ملفات الـ Inodes، قراءة الـ Superblock، استرجاع الملفات المحذوفة Sleuth Kit, Autopsy
تحليل سجلات النظام تحليل سجلات المصادقة secure وسجلات الـ Syslog وجداول التدقيق auditd LogParser, Grep/RegEx
إعادة بناء خطوط الزمن إنشاء Timeline دقيق للملفات والعمليات المستحدثة وقت الهجوم Plaso, log2timeline
تحليل الذاكرة (Memory) فحص الذاكرة العشوائية رام لاستخراج اتصالات الشبكة النشطة وكلمات المرور Volatility 3

📖 كتب مقترحة للمرحلة: Practical Linux Forensics - Bruce Nikkel

▶ المرحلة 5: خفايا النواة والبرمجة (Kernel & Internals)
مستوى النخبة

الهدف: فهم كيف تعمل النواة في إدارة الموارد، برمجة النظام بالـ C، وتحليل الأداء باستخدام eBPF.

الموضوع المحتوى والتطبيقات الأدوات والتقنيات
إدارة العمليات والذاكرة بنية الـ Process Descriptor، آليات الـ Virtual Memory، الـ Scheduler Linux Kernel Code
برمجة النظام (Syscalls) استخدام الـ System Calls مباشرة، الـ IPC، الـ Threading والمزامنة GNU C Library
تحليل الأداء المتقدم كتابة سكريبتات eBPF لتتبع استدعاءات النظام ومتابعة أداء الأنوية بالملي ثانية eBPF, bpftrace, perf
برمجة الـ Kernel Modules كتابة وتثبيت Loadable Kernel Modules لتوسيع وظائف النواة أو كتابة برامج تعريفية Kbuild, insmod, rmmod

📖 كتب مقترحة للمرحلة: Linux Kernel Development - Robert Love
