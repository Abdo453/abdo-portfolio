Linux Commands Cheat Sheet / مرجع الأوامر السريع

AR

جدول مصنف لأهم الأوامر والبارامترات التي يستخدمها باحث الثغرات ومختبر الاختراق أثناء التعامل مع خوادم لينكس.

EN

A categorized table containing the most critical commands and parameters used by researchers during Linux host audits.

Category / الفئة Command / الأمر Description (EN) الوصف (AR)
File Operations cat [file] Display file contents عرض محتويات الملف
File Operations grep "[pattern]" [file] Search for matching strings البحث عن نصوص مطابقة داخل ملف
System Audit uname -a Print kernel version details عرض تفاصيل إصدار النواة ونوع النظام
System Audit ps aux List running processes عرض العمليات والبرامج النشطة حالياً
Network netstat -tulnp List listening ports and sockets عرض المنافذ المفتوحة والمستمعة على السيرفر
Network ss -antp List active network connections عرض اتصالات الشبكة النشطة
Permissions chmod +x [file] Make file executable جعل الملف قابلاً للتشغيل كبرنامج
Permissions chown root:root [file] Change file owner to root تغيير مالك ومجموعة الملف إلى root
