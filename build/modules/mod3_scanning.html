🧭 1. Host Discovery — مين شغال على الشبكة؟

📡 Ping Sweep

يبعت ICMP لكل IP في الـ Range ويشوف مين بيرد.

nmap -sn 192.168.1.0/24

-sn = Ping Scan بدون فحص بورتات

⚡ ARP Scan (أقوى في الشبكة المحلية)

يشتغل حتى لو Ping مقفول — لأنه يتواصل على مستوى MAC Address.

netdiscover -r 192.168.1.0/24
arp-scan -l

🌐 2. أنواع Port Scanning — اعرف كل نوع وإمتى تستخدمه

🔥 TCP SYN Scan (Stealth Scan) — الأهم في الامتحان ⭐ الأكثر استخداماً

يبعت SYN فقط، لو جاء SYN-ACK = البورت مفتوح، ثم يبعت RST لإنهاء الاتصال بدون إكماله. سريع + أقل كشفاً.

nmap -sS 192.168.1.1
✅ سريع جداً
✅ صعب اكتشافه
✅ لا يكمل الاتصال
⚠️ يحتاج صلاحيات root
TCP Connect Scan (-sT)
nmap -sT 192.168.1.1

اتصال كامل (3-way handshake). دقيق لكن سهل يتكشف في الـ Logs.

لما مش معاك root access
FIN Scan (-sF)
nmap -sF 192.168.1.1

يبعت FIN Flag فقط. البورت المفتوح = لا رد. البورت المقفول = RST.

لتجاوز بعض أنواع الـ Firewalls
XMAS Scan 🎄 (-sX)
nmap -sX 192.168.1.1

يبعت FIN + URG + PSH معاً. لتجاوز الـ Firewalls لكن لا يعمل على Windows.

يعمل بشكل جيد على Linux targets
UDP Scan (-sU)
nmap -sU 192.168.1.1

لاكتشاف خدمات UDP: DNS(53), SNMP(161), TFTP(69). بطيء جداً.

يستغرق وقتاً طويلاً — استخدمه بحكمة

🧠 3. أوامر Nmap المتقدمة

🔍 OS Detection
nmap -O 192.168.1.1

يحاول يتعرف على نظام التشغيل (Windows / Linux) من خلال تحليل استجابات الـ TCP/IP.

⚙️ Service Detection
nmap -sV 192.168.1.1

يجيب اسم الخدمة وإصدارها (مثال: Apache 2.4.49، OpenSSH 7.6p1).

📜 NSE Scripts
nmap --script vuln 192.168.1.1
nmap --script http-enum 192.168.1.1

مكتبة سكريبتات ضخمة في /usr/share/nmap/scripts — فحص ثغرات، enum، وأكثر.

💣 Full Aggressive Scan
nmap -A 192.168.1.1

يجمع كل شيء: Port scan + OS + Service + Scripts. ضجيج عالي جداً (Noisy).

🎯 Port Range Options
nmap -p 80,443 192.168.1.1 # بورتات محددة
nmap -p- 192.168.1.1 # كل 65535 بورت
nmap -T4 192.168.1.1 # سرعة عالية (0-5)
🖥️ Netcat — Manual Check
nc -nv 192.168.1.1 80

اختبار يدوي هل البورت مفتوح — مفيد جداً في Banner Grabbing والـ OSCP.

🥷 4. تجاوز الـ Firewall و IDS

🔀 Source Port Spoofing
nmap --source-port 80 192.168.1.1

يخلي الترافيك يبدو كأنه قادم من Web Server.

📦 Fragmentation
nmap -f 192.168.1.1

يقسم الـ Packets لأجزاء صغيرة — صعب اكتشافه من الـ IDS.

🎭 Decoy Attack
nmap -D RND:10 192.168.1.1

يخلط Scan الحقيقي مع 10 IPs وهمية — محير لنظام الحماية!

🔄 MAC Spoofing
nmap --spoof-mac 0 192.168.1.1

يغير الـ MAC Address عشوائياً لتجنب الكشف.

🏆 أفضل Scan عملي — احفظه!

nmap -sS -sV -O -T4 -p- 192.168.1.1
