Module 04 Deep Extraction

⚡ أولاً: يعني إيه Enumeration؟

هو إنك تعمل اتصال مباشر بالهدف وتسأله أسئلة (Queries) علشان يطلعلك بيانات 🎯

📌 المعلومات المستهدفة:

  • 👤 Users (المستخدمين)
  • 🔑 Passwords (كلمات المرور)
  • 📁 Shares (المجلدات المشتركة)
  • ⚙️ Services details (تفاصيل الخدمات)

🔥 الفرق عن Scanning:

  • Scanning: “فيه إيه؟ الباب ده مفتوح؟”
  • Enumeration: “هات التفاصيل الدقيقة! مين جوه؟”

🌐 1) HTTP Enumeration (مواقع الويب)

🧠 الأساس: GET vs POST

🔹 GET Request
GET /index.php?id=1

👉 البيانات بتظهر في الـ URL (أقل أماناً)

🔹 POST Request
POST /login username=admin&password=123

👉 البيانات مخفية في الـ Body (أكثر أماناً)

🔍 أدوات HTTP Enumeration

1. Nikto (مهم جدًا)
nikto -h http://192.168.1.1

👉 بيكشف: Vulnerabilities، ملفات خطيرة، إعدادات غلط.

2. Nmap Scripts
nmap --script=http-enum 192.168.1.1

👉 يطلع: directories، صفحات مخفية.

3. Directory Bruteforce
dirb http://192.168.1.1
wfuzz -c -w /usr/share/wordlists/dirb/common.txt http://192.168.1.1/FUZZ

👉 بيكتشف: admin panels، hidden APIs.

4. SecLists (كنز 🔥)
/usr/share/seclists

👉 فيها: قوائم (Wordlists) جاهزة للـ brute force، directories، passwords.

5. Burp Suite

👉 يستخدم في: اعتراض الطلبات (Intercept)، تعديلها، واكتشاف الثغرات (Web hacking و SQL Injection).

🖥️ 2) SMB Enumeration (ويندوز)

1. enum4linux
enum4linux 192.168.1.1

👉 يطلع: Users، Shares، Password policy.

2. Nmap SMB Scripts
nmap --script=smb-enum* 192.168.1.1
3. Metasploit SMB
use auxiliary/scanner/smb/smb_enumshares

🌐 3) NetBIOS Enumeration

nmap -sU -p 137 --script=nbstat 192.168.1.1

👉 يطلع: اسم الجهاز، domain.

📁 4) NFS Enumeration

use auxiliary/scanner/nfs/nfsmount

👉 باستخدام Metasploit، يطلع: Shared folders.

🔐 5) SSH Enumeration

use auxiliary/scanner/ssh/ssh_version

👉 باستخدام Metasploit، يجيب: إصدار SSH.

🎯 6) Banner Grabbing (مهم جدًا)

👉 فكرة بسيطة: تاخد “البانر” من السيرفر = معلومات عن الخدمة والإصدار.

nc -nv 192.168.1.1 80

👉 ممكن يرجع: Apache version، OS.

💣 7) البحث عن Exploits

  • محلي (Kali): searchsploit apache
  • أونلاين: Exploit-DB، GitHub

🧠 8) UDP Enumeration

nmap -sU 192.168.1.1

👉 مهم لخدمات زي: DNS, SNMP.

⚠️ ملاحظات مهمة

  • Enumeration = مرحلة حساسة جدًا ومزعجة (Noisy) 🔥.
  • غالبًا تحتاج: Brute force و Password guessing.
  • خطر: ممكن تتكشف بسهولة من أنظمة الـ IDS/IPS.

💡 أفضل Workflow عملي

  1. Scan: nmap -sS -sV -O 192.168.1.1
  2. HTTP: nikto -h http://IP
  3. SMB: enum4linux IP
  4. Directories: dirb http://IP
