📋 Module 4: Enumeration (التعداد)

بعد فتح الأبواب، حان وقت التفتيش — سحب بيانات حقيقية وحساسة من الهدف مباشرةً.

Module 04 Data Extraction OSCP Critical

🧠 ما هو الـ Enumeration؟

الـ Enumeration هو إنشاء اتصال مباشر بالهدف وسؤاله بشكل نشط للحصول على بيانات تفصيلية دقيقة. الفرق عن الـ Scanning أنك مش بس بتسأل "مفتوح ولا لأ؟" — دلوقتي بتسأل "إيه موجود جواه؟"

📡 Scanning يقول:
  • "البورت 80 مفتوح"
  • "البورت 445 مفتوح"
  • "البورت 22 مفتوح"
🔥 Enumeration يقول:
  • "على البورت 80: Apache 2.4.49 وفيه /admin"
  • "على 445: Users: admin, john. Share: C$"
  • "على 22: OpenSSH 7.6p1 يقبل Password Auth"

🌐 1. HTTP Enumeration (مواقع الويب)

GET Request
GET /index.php?id=1 HTTP/1.1

البيانات ظاهرة في الـ URL — سهل اكتشافه في الـ Logs.

POST Request
POST /login HTTP/1.1
username=admin&password=123

البيانات في الـ Body — أكثر أماناً ولكن يمكن اعتراضه بـ Burp Suite.

🔍 أدوات HTTP Enumeration:
Nikto — فحص شامل
nikto -h http://192.168.1.1

يكشف الثغرات المعروفة، الملفات الخطيرة، الإعدادات الخاطئة (Outdated, Misconfig).

dirb + wfuzz — اكتشاف المسارات
dirb http://192.168.1.1
wfuzz -c -w /usr/share/wordlists/dirb/common.txt http://IP/FUZZ

يكتشف admin panels, /backup, /config, hidden APIs.

Burp Suite — الأقوى

يعترض الطلبات ويسمح بتعديلها — ضروري في Web Hacking و SQL Injection. يعمل كـ Proxy بين متصفحك والموقع.

أكثر أداة استخداماً في Bug Bounty!
SecLists — كنز الـ Wordlists 🔥
/usr/share/seclists/

يحتوي على آلاف الـ Wordlists الجاهزة: Directories, Passwords, Usernames, Subdomains, Fuzzing.

🖥️ 2. SMB Enumeration (ويندوز)

الـ SMB (Port 445) بروتوكول مشاركة الملفات في Windows — كنز من البيانات إذا كان مفتوحاً!

enum4linux — الأداة الكل في واحد
enum4linux 192.168.1.1
enum4linux -a 192.168.1.1 # كل شيء

يجيب: Users, Groups, Shares, Password Policy, OS Info.

Nmap SMB Scripts
nmap --script=smb-enum* 192.168.1.1
nmap --script=smb-vuln-ms17-010 IP # EternalBlue!

يفحص مشاركات، ثغرات، وإعدادات SMB.

Metasploit SMB
use auxiliary/scanner/smb/smb_enumshares
use auxiliary/scanner/smb/smb_enumusers

يعطيك قائمة بالمشاركات والمستخدمين بشكل منظم.

🔌 3. Enumeration للبروتوكولات الأخرى

🌐 NetBIOS (Port 137)
nmap -sU -p 137 --script=nbstat 192.168.1.1

يجيب: اسم الجهاز، اسم الـ Domain، MAC Address.

📁 NFS (Network File System)
showmount -e 192.168.1.1
use auxiliary/scanner/nfs/nfsmount

يجيب قائمة الـ Shared Folders — قد تكون مفتوحة للـ Mount!

🔐 SSH (Port 22)
use auxiliary/scanner/ssh/ssh_version
nmap -sV -p22 192.168.1.1

يجيب إصدار SSH — إصدارات قديمة قد تكون عرضة للثغرات.

📡 SNMP (Port 161 UDP)
snmpwalk -v1 -c public 192.168.1.1
use auxiliary/scanner/snmp/snmp_enum

بروتوكول إدارة الشبكات — غالباً بـ Community String افتراضي "public"!

🎯 4. Banner Grabbing — اقرأ بطاقة تعريف الخدمة

الـ Banner هو الرسالة الأولى التي يرسلها الخادم عند الاتصال — تحتوي على اسم الخدمة وإصدارها!

nc -nv 192.168.1.1 80 # HTTP Banner
nc -nv 192.168.1.1 21 # FTP Banner
nc -nv 192.168.1.1 25 # SMTP Banner
# نتيجة مثال:
220 vsftpd 2.3.4
→ ثغرة Backdoor معروفة! 🔥
💣 البحث عن الـ Exploits بعد معرفة الإصدار:
searchsploit vsftpd 2.3.4
searchsploit apache 2.4.49

أيضاً تقدر تبحث على exploit-db.com و GitHub لو مش لاقي في searchsploit.

🏆 Workflow عملي مثالي

  1. Scan: nmap -sS -sV -O 192.168.1.1
  2. HTTP: nikto -h http://IP ثم dirb http://IP
  3. SMB: enum4linux IP
  4. Banner: nc -nv IP [port]
  5. Exploit Search: searchsploit [service] [version]
