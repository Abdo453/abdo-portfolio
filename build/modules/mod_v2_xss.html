💉 Cross-Site Scripting (XSS)

ثغرة تسمح للمهاجم بحقن أكواد JavaScript خبيثة داخل صفحات ويب يثق بها المستخدمون الآخرون.

📖 النظري (Theory)

التأثير (Impact)
  • سرقة الجلسات (Session Hijacking)
  • تغيير محتوى الصفحة (Defacement)
  • إعادة التوجيه لمواقع خبيثة
  • Keylogging

🎯 الأنواع (Types)

Reflected XSS

البايلود ينعكس في نفس الاستجابة (مثل رابط بحث). يتطلب تفاعل الضحية.

الخطورة: High
Stored XSS

البايلود يُحفظ في قاعدة بيانات الموقع (مثل تعليق). يضرب أي ضحية تفتح الصفحة.

الخطورة: Critical
DOM-Based XSS

التنفيذ يحدث بالكامل في متصفح الضحية بسبب تعامل الـ JavaScript بشكل غير آمن مع الـ DOM.

الخطورة: High
