Level 1: Security Basics

فهم المتغيرات، الدوال، والتعامل مع الملفات والنصوص بعقلية الهاكر.

معالجة النصوص (String Manipulation)

في الأمن السيبراني، استخراج البيانات المخفية (Parsing) هو مهارة أساسية. بايثون تجعل التعامل مع النصوص سهلاً جداً.

strings_hack.py
import re # Extracting hidden endpoints from a JS file js_content = "var api_url = '/api/v1/users'; fetch('/internal/debug');" endpoints = re.findall(r"/[a-zA-Z0-9_/-]+", js_content) for endpoint in endpoints: print(f"[*] Found Hidden Endpoint: {endpoint}")
[*] Found Hidden Endpoint: /api/v1/users [*] Found Hidden Endpoint: /internal/debug
🎯 Mini-Challenge

استخدم `re.findall` لاستخراج جميع عناوين الـ IP من ملف السجل (Log file).

Level 2: Networking

كيف تتحدث مع السيرفرات: Sockets و Requests.

مكتبة Requests (تخطي الحمايات)

مكتبة `requests` هي سلاحك الأول في الـ Web Hacking. هنا مثال على محاولة تخطي 403 Forbidden بتغيير الـ Headers.

bypass_403.py
import requests url = "https://target.com/admin" headers = { "X-Forwarded-For": "127.0.0.1", "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)" } print("[*] Sending request to /admin with local IP spoofing...") response = requests.get(url, headers=headers, allow_redirects=False) if response.status_code == 200: print("[+] BYPASS SUCCESS! Admin Panel accessed.") else: print(f"[-] Failed. Status Code: {response.status_code}")

Level 3: Automation

أتمتة المهام وبناء أدوات استطلاع (Recon) مصغرة.

Automated Subdomain Resolver

عندما تحصل على آلاف النطاقات الفرعية من Subfinder، تحتاج إلى سكربت سريع لفلترة النطاقات الحية.

resolve_subs.py
import socket import concurrent.futures subdomains = ["api.target.com", "dev.target.com", "fake123.target.com"] def check_domain(domain): try: ip = socket.gethostbyname(domain) return f"[+] Alive: {domain} -> {ip}" except socket.gaierror: return f"[-] Dead : {domain}" with concurrent.futures.ThreadPoolExecutor(max_workers=10) as executor: results = executor.map(check_domain, subdomains) for res in results: print(res)
[+] Alive: api.target.com -> 104.21.5.12 [+] Alive: dev.target.com -> 172.67.12.3 [-] Dead : fake123.target.com
