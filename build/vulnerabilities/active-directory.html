اختبار اختراق الـ Active Directory

Active Directory Pentesting

CRITICAL AD Security
AR

Active Directory (AD) هو نظام إدارة الهويات والوصول الرئيسي في شبكات المؤسسات والشركات الكبرى. تركز عمليات اختبار اختراق الـ AD على استغلال البروتوكولات الأمنية القديمة، وسرقة الـ Hashes لكلمات المرور، والبحث عن مسارات ترقية الصلاحيات للوصول لـ Domain Admin.

EN

Active Directory (AD) is the enterprise directory service developed by Microsoft. Active Directory pentesting focuses on identifying security misconfigurations, extracting credential hashes, and mapping paths to achieve Domain Administrator privileges.

أشهر ثغرات وهجمات الـ Active Directory / Attack Types

AD Enumeration

جمع بيانات المستخدمين والمجموعات والصلاحيات والأجهزة داخل الشبكة.

Kerberoasting

طلب بطاقات الخدمة (TGS tickets) للمستخدمين الذين يمتلكون قيم SPN ومحاولة فك تشفيرها خارجياً للحصول على كلمات المرور.

AS-REP Roasting

استهداف حسابات لا تتطلب مصادقة مسبقة لـ Kerberos وسرقة الـ Hashes الخاصة بها مباشرة.

Pass-the-Hash (PtH)

استخدام الـ NTLM Hash لتسجيل الدخول للأجهزة دون الحاجة لمعرفة كلمة المرور النصية.

Golden & Silver Tickets

تزوير بطاقات Kerberos كاملة الصلاحية للبقاء الدائم بالشبكة (Persistence).

BloodHound Analysis

تحليل مسارات الهجوم الشجرية للوصول لمدير النطاق (Domain Admin Path).

أوامر الاختبار والتحليل / AD Commands

# Python collector for BloodHound enumeration
bloodhound-python -d targetdomain.local -u domain_user -p domain_password -c All -ns 10.10.10.100

# Dumping hashes from Domain Controller with Secretsdump
impacket-secretsdump targetdomain/admin_user:admin_pass@10.10.10.100

# Performing Kerberoasting
impacket-GetUserSPNs targetdomain.local/domain_user:domain_password -request -dc-ip 10.10.10.100

الأدوات / Tools

BloodHound Impacket Suite (secretsdump, psexec) CrackMapExec / NetExec Mimikatz PowerView
