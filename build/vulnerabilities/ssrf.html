SSRF — تزوير طلبات من جانب السيرفر

SSRF — Server-Side Request Forgery

CRITICAL OWASP A10
AR

SSRF تجبر السيرفر على إرسال HTTP requests لعناوين داخلية أو خارجية يختارها المهاجم — للوصول لـ cloud metadata، خدمات داخلية، أو bypass firewalls.

EN

SSRF forces the server to send HTTP requests to attacker-chosen internal or external addresses — accessing cloud metadata, internal services, or bypassing firewalls.

أنواع SSRF / SSRF Types

Response يظهر مباشرة — scan الشبكة الداخلية

Cloud Metadata

file:// Protocol

gopher://
