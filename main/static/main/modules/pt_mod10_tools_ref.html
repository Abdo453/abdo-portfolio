🔥 10. Essential Tools Reference

دليل شامل لأهم الأدوات المستخدمة في اختبار الاختراق واكتشاف الثغرات

💡 تلميح: تعتبر هذه القائمة المرجع الأساسي لأدوات اختبار الاختراق ومكافآت الثغرات. يعتمد المحترفون على دمج هذه الأدوات معاً لإنشاء سير عمل متكامل (Pipelines).

10.1 Reconnaissance Tools

أدوات الاستطلاع (Reconnaissance) هي المرحلة الأولى والأهم في أي عملية هجوم. تُستخدم لجمع أكبر قدر ممكن من المعلومات حول الهدف (مثل النطاقات الفرعية، الأجهزة الحية، والمسارات) قبل التفاعل المباشر.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
subfinder Passive subdomain enumeration أداة سريعة للبحث السلبي عن النطاقات الفرعية (Passive Recon) تعتمد على جمع المعلومات من مصادر علنية (OSINT) دون التفاعل المباشر مع خوادم الهدف، مما يجعلها خفية وفعالة. go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
amass Comprehensive subdomain enumeration منصة قوية وشاملة لجمع المعلومات واكتشاف الأصول التابعة للهدف باستخدام تقنيات متعددة (سلبية ونشطة)، بالإضافة إلى رسم خرائط للشبكات. go install github.com/owasp-amass/amass/v4/...@master
assetfinder Subdomain discovery (crt.sh + Facebook CT) أداة خفيفة تركز على إيجاد النطاقات الفرعية عبر مصادر محددة مثل سجلات شهادات الأمان (crt.sh) وغيرها. go install github.com/tomnomnom/assetfinder@latest
httpx Live host probing مجموعة أدوات لبروتوكول HTTP تتيح لك التحقق السريع من حالة الخوادم الحية (Live hosts) واستخراج معلومات مثل أكواد الحالة (Status Codes) والتقنيات المستخدمة. go install github.com/projectdiscovery/httpx/cmd/httpx@latest
shuffledns MassDNS wrapper for subdomain brute force أداة تغليف (Wrapper) لبرنامج MassDNS، تسهل عمليات التخمين العنيف (Brute-forcing) للنطاقات الفرعية وتحليل سجلات DNS بسرعة هائلة. go install github.com/projectdiscovery/shuffledns/cmd/shuffledns@latest
massdns High-speed DNS resolver محلل DNS (Resolver) عالي الأداء يمكنه معالجة ملايين الطلبات في ثوانٍ، يُستخدم في عمليات المسح واسعة النطاق لضمان أقصى سرعة. git clone https://github.com/blechschmidt/massdns.git
puredns Fast domain resolver with wildcard filtering أداة حديثة ومطورة تعتمد على MassDNS لحل النطاقات وفلترة النتائج الوهمية (Wildcard filtering) لضمان دقة النطاقات المكتشفة وتجنب الإيجابيات الكاذبة. go install github.com/d3mondev/puredns/v2@latest
altdns Subdomain permutation generator أداة تقوم بتوليد تنويعات وتباديل (Permutations) لأسماء النطاقات الفرعية لمحاولة اكتشاف نطاقات مخفية لم تظهر في البحث العادي، مفيدة جداً لتوسيع نطاق الهجوم. pip install py-altdns
gotator Advanced subdomain permutation أداة مطورة بلغة Go لإنشاء تنويعات لأسماء النطاقات (Permutations) بسرعة وكفاءة عالية، تقدم أداءً محسناً مقارنة بـ altdns. go install github.com/Josue87/gotator@latest
dnsx DNS toolkit مجموعة أدوات DNS متعددة الأغراض تتيح الاستعلام المتعدد، وفلترة النطاقات النشطة (Alive filtering)، واكتشاف سجلات CNAME و A بسرعة خيالية. go install github.com/projectdiscovery/dnsx/cmd/dnsx@latest
gau URL fetcher (AlienVault, Wayback, etc.) أداة جلب الروابط التاريخية (Fetch URLs) من مصادر متعددة مثل AlienVault و Wayback Machine، مفيدة لاكتشاف مسارات ونقاط نهاية (Endpoints) قديمة قد تكون غير محمية. go install github.com/lc/gau/v2/cmd/gau@latest
waybackurls Wayback Machine URL fetcher أداة بسيطة تركز فقط على سحب جميع الروابط المؤرشفة لنطاق معين من موقع أرشيف الإنترنت (Wayback Machine). go install github.com/tomnomnom/waybackurls@latest
katana Web crawler زاحف ويب (Web Crawler) حديث وسريع يستخدم لاكتشاف المسارات والروابط المخبأة في تطبيقات الويب من خلال محاكاة تصفح المستخدم بذكاء. go install github.com/projectdiscovery/katana/cmd/katana@latest
nuclei Vulnerability scanner محرك فحص ثغرات يعتمد على قوالب (Templates) مخصصة، مصمم لإرسال طلبات محددة واكتشاف الثغرات المعروفة، والشهادات المنتهية، والأخطاء البرمجية بدقة عالية وسرعة فائقة. go install github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest
naabu Port scanner أداة مسح منافذ (Port Scanner) موثوقة وسريعة مكتوبة بلغة Go، تركز على اكتشاف المنافذ المفتوحة بصمت وبكفاءة مع التوافقية للربط مع أدوات أخرى. go install github.com/projectdiscovery/naabu/v2/cmd/naabu@latest

10.2 Content Discovery Tools

أدوات اكتشاف المحتوى تُستخدم للبحث عن المجلدات المخفية، الملفات الحساسة، والمعلمات (Parameters) التي لم يتم الإعلان عنها، مما يوسع مساحة الهجوم بشكل كبير.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
ffuf Fast web fuzzer أداة فازينج (Fuzzer) سريعة جداً للويب، تستخدم لاكتشاف الملفات والمجلدات المخفية، وتخمين المعلمات (Parameters)، والبحث عن الثغرات بمرونة عالية. go install github.com/ffuf/ffuf/v2@latest
feroxbuster Rust-based content discovery أداة لاكتشاف المحتوى المخفي (Directory Brute-forcer) مبنية بلغة Rust، تتميز بالسرعة المهولة ودعم المسح المتكرر (Recursive) للمجلدات واكتشاف المسارات تلقائياً. cargo install feroxbuster
gobuster Directory/file/DNS brute forcer أداة شهيرة وسريعة لتخمين المسارات، والملفات، والنطاقات الفرعية، وحتى خوادم DNS التابعة للهدف. يعتمد عليها الكثير لسهولتها. go install github.com/OJ/gobuster/v3@latest
dirsearch Python-based directory brute forcer أداة كلاسيكية مبنية بـ Python لاكتشاف المسارات والملفات المخبأة في خوادم الويب، تدعم تخصيص القواميس (Wordlists) وإخراج النتائج بتنسيقات متعددة. git clone https://github.com/maurosoria/dirsearch.git
arjun HTTP parameter discovery أداة متخصصة في اكتشاف المعلمات المخفية (Hidden HTTP Parameters) في طلبات الويب، مفيدة جداً في البحث عن ثغرات XSS أو SSRF أو SQLi التي لا تظهر في الواجهة. pip install arjun
paramspider Parameter discovery from URLs أداة تعتمد على مصادر الأرشيف لجمع المعلمات (Parameters) المرتبطة بالنطاق، مما يوفر الكثير من الوقت بدلاً من البحث عنها يدوياً. git clone https://github.com/devanshbatham/ParamSpider.git
xnLinkFinder Link finder from JS files أداة مخصصة لتحليل ملفات JavaScript واستخراج الروابط (Links) ونقاط النهاية (Endpoints) التي قد تكون مخفية عن المستخدم العادي. git clone https://github.com/xnl-h4ck3r/xnLinkFinder.git
jsluice JavaScript analysis أداة متقدمة لتحليل شفرات JavaScript واستخراج الروابط، المسارات، والأسرار (Secrets) الدفينة، وتعتمد على بناء شجرة لغوية (AST) لضمان دقة التحليل. go install github.com/BishopFox/jsluice/...
burp suite Web proxy and scanner المنصة الأساسية والأقوى لاختبار اختراق تطبيقات الويب. تعمل كوكيل (Proxy) لاعتراض وتعديل الطلبات، وتحتوي على أدوات فحص وتخمين متعددة لا غنى عنها لأي مختبر اختراق. https://portswigger.net/burp/communitydownload
owasp zap Web proxy and scanner البديل المجاني ومفتوح المصدر لـ Burp Suite، يقدم ميزات مشابهة لاعتراض الطلبات وفحص الثغرات الآلي ويُستخدم بشكل واسع في بيئات CI/CD. https://www.zaproxy.org/download/
nikto Web server scanner أداة فحص كلاسيكية لخوادم الويب، تقوم باختبار الخوادم بحثاً عن الملفات الخطيرة، والبرمجيات القديمة، والتكوينات الخاطئة. apt install nikto

10.3 Port Scanning & Network Tools

هذه الأدوات أساسية لاكتشاف المنافذ المفتوحة والخدمات التي تعمل على الخوادم، مما يوفر نقطة انطلاق لاستغلال الشبكات.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
nmap Network scanner الأداة المعيارية الأولى في مسح الشبكات لاكتشاف الأجهزة الحية، المنافذ المفتوحة، الخدمات، وأنظمة التشغيل، وتدعم تشغيل نصوص برمجية (NSE) لاكتشاف الثغرات المتقدمة. apt install nmap
masscan High-speed port scanner أسرع أداة لمسح المنافذ (Port Scanner) قادرة على مسح الإنترنت بالكامل في دقائق معدودة عبر إرسال ملايين الحزم في الثانية، مثالية للنطاقات الواسعة. git clone https://github.com/robertdavidgraham/masscan.git
rustscan Fast port scanner أداة مسح منافذ حديثة وسريعة جداً مبنية بـ Rust، تقوم بتحويل النتائج تلقائياً إلى Nmap لإجراء فحص أعمق، مما يختصر وقت الفحص بشكل كبير. cargo install rustscan
naabu Go port scanner أداة مسح موثوقة تركز على الموثوقية العالية والأداء القوي، متوافقة بشدة مع بقية أدوات ProjectDiscovery. See above
wireshark Network protocol analyzer محلل حزم الشبكة (Packet Analyzer) الرائد عالمياً، يسمح بمراقبة حركة المرور (Traffic) وتحليل بروتوكولات الشبكة على مستوى دقيق جداً باستخدام واجهة رسومية. apt install wireshark
tcpdump Packet analyzer أداة سطر أوامر قوية لالتقاط وتحليل حزم البيانات المارة عبر الشبكة، تُستخدم غالباً في الخوادم البعيدة التي لا تحتوي على واجهة رسومية. apt install tcpdump
netcat Network utility أداة الشبكات متعددة الاستخدامات، تُعرف بـ "سكين الجيش السويسري للشبكات"، تُستخدم لقراءة وكتابة البيانات عبر اتصالات TCP/UDP وإنشاء جلسات اتصال عكسية (Reverse Shells). apt install netcat
impacket Python network protocols مجموعة أدوات بـ Python للتعامل مع بروتوكولات شبكات Windows مثل SMB، WMI، و Kerberos. تعتبر هذه المجموعة حجر الأساس في اختبارات اختراق بيئة Active Directory. pip install impacket

10.4 Web Application Testing Tools

مجموعة من الأدوات المخصصة لاكتشاف واستغلال ثغرات تطبيقات الويب الشهيرة مثل حقن قواعد البيانات (SQLi) والبرمجة عبر المواقع (XSS).

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
burp suite Web proxy/scanner أقوى منصة لاعتراض وتحليل طلبات الويب، تسمح لك بالتحكم الكامل في حركة المرور لتطبيق الويب (Traffic) لاكتشاف واستغلال الثغرات المختلفة. See above
sqlmap SQL injection automation أداة أوتوماتيكية بالكامل لاكتشاف واستغلال ثغرات حقن قواعد البيانات (SQL Injection)، قادرة على استخراج قواعد البيانات والوصول للنظام. git clone https://github.com/sqlmapproject/sqlmap.git
ghauri Modern SQL injection tool أداة حديثة تعتمد على Python لاكتشاف واستغلال ثغرات حقن قواعد البيانات (SQLi)، وتتميز بقدرتها على تجاوز بعض الحمايات الحديثة بكفاءة. pip install ghauri
dalfox XSS scanner أداة فحص سريعة وقوية جداً مخصصة لاكتشاف ثغرات XSS، وتدعم التحليل المتقدم للمتغيرات (Parameters) وحقن الحمولات (Payloads) تلقائياً. go install github.com/hahwul/dalfox/v2@latest
xsstrike XSS detection suite مجموعة أدوات متكاملة لاكتشاف XSS، تعتمد على تحليل الاستجابة وتوليد حمولات (Payloads) ديناميكية لتجاوز جدران الحماية (WAF) بذكاء. git clone https://github.com/s0md3v/XSStrike.git
wafw00f WAF detection أداة تُستخدم لتحديد نوع جدار الحماية لتطبيقات الويب (WAF) الذي يحمي الهدف، مما يساعد في وضع خطة مناسبة لتجاوز هذه الحماية. pip install wafw00f
whatweb Technology fingerprinting أداة لجمع بصمات التقنيات (Fingerprinting) لمعرفة نوع خادم الويب، نظام إدارة المحتوى (CMS)، إصدارات البرمجيات، ورؤوس HTTP. apt install whatweb
gowitness Screenshotting أداة لأخذ لقطات شاشة (Screenshots) جماعية لتطبيقات الويب على نطاق واسع باستخدام متصفح Headless، لتسهيل التعرف البصري على الأهداف. go install github.com/sensepost/gowitness@latest
aquatone Screenshotting أداة مشابهة تقوم بأخذ لقطات شاشة للنطاقات المكتشفة وتجميعها في تقرير HTML منظم ومرئي لتسريع عملية الاستكشاف وتحليل مساحة الهجوم. go install github.com/michenriksen/aquatone@latest

10.5 API Testing Tools

تتطلب واجهات برمجة التطبيقات (APIs) أدوات متخصصة للتعامل مع البيانات المهيكلة (مثل JSON و GraphQL) لاختبار أمانها بفعالية.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
postman API testing platform منصة شهيرة لتطوير واختبار واجهات برمجة التطبيقات (APIs)، تتيح إرسال طلبات معقدة وتنظيم المجموعات (Collections) لتسهيل الفحص اليدوي. https://www.postman.com/downloads/
insomnia API testing client عميل API بديل لـ Postman يتميز بواجهة نظيفة وسريعة، يدعم REST و GraphQL و gRPC، ومفيد جداً في بناء طلبات الاختبار اليدوية للـ APIs. https://insomnia.rest/download
httpie Command-line HTTP client عميل HTTP يعمل عبر سطر الأوامر، يعتبر بديلاً أسهل وأكثر قابلية للقراءة من curl، مصمم للتعامل مع واجهات API والـ JSON بسهولة. apt install httpie
jwt_tool JWT testing toolkit مجموعة أدوات متخصصة في اختبار رموز JWT (JSON Web Tokens)، تتيح فك التشفير، التلاعب بالبيانات (Claims)، ومحاولة تخطي آليات التحقق من التوقيع. git clone https://github.com/ticarpi/jwt_tool.git
inql GraphQL testing أداة فحص واختبار لـ GraphQL يمكن استخدامها بشكل مستقل أو كإضافة لـ Burp Suite، تسحب المخططات (Schemas) لتسهيل استكشاف وبناء الاستعلامات. pip install inql
graphqlmap GraphQL exploitation أداة مخصصة للتفاعل مع مسارات GraphQL واستغلالها، تدعم حقن الثغرات الشائعة (مثل NoSQLi، XSS، SSRF) مباشرة في استعلامات GraphQL. git clone https://github.com/swisskyrepo/GraphQLmap.git

10.6 Cloud & Infrastructure Tools

مع انتقال الشركات إلى الحوسبة السحابية، أصبحت هذه الأدوات حتمية لاكتشاف الموارد السحابية غير المحمية والمفاتيح المسربة.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
s3scanner S3 bucket enumeration أداة متخصصة في فحص حزم التخزين السحابية (S3 Buckets) الخاصة بـ AWS لاكتشاف الحزم المفتوحة للعامة أو ذات الصلاحيات الضعيفة التي يمكن تسريب بيانات منها. pip install s3scanner
cloud_enum Multi-cloud enumeration أداة قوية للبحث في السحابات العامة المتعددة (AWS، Azure، GCP) لاكتشاف الموارد، الملفات، والأصول غير المحمية التابعة لهدف معين. git clone https://github.com/initstring/cloud_enum.git
gcpbucketbrute GCP bucket enumeration أداة مخصصة لتخمين واكتشاف حزم التخزين (Storage Buckets) المخفية أو غير الآمنة في بيئة Google Cloud Platform. git clone https://github.com/RhinoSecurityLabs/GCPBucketBrute.git
trufflehog Secret scanning أداة متقدمة للبحث العميق في مستودعات الأكواد (Repositories) وتاريخ الالتزامات (Commits) لاكتشاف المفاتيح السرية، كلمات المرور، ورموز الوصول المسربة. pip install truffleHog
gitleaks Secret scanning أداة سريعة للبحث التلقائي عن الأسرار والمفاتيح (Secrets/Tokens) في مشاريع Git، تُستخدم بكثرة في عمليات التطوير (CI/CD) وكذلك من قبل المهاجمين لاستخراج الأسرار. go install github.com/zricethezav/gitleaks/v8@latest
gitdorker GitHub dorking أداة تستخدم تقنية البحث المتقدم (Dorking) للبحث في GitHub عن معلومات حساسة متعلقة بالهدف عبر آلاف الاستعلامات المخصصة. git clone https://github.com/obheda12/GitDorker.git
asnmap ASN enumeration أداة لجمع معلومات نظام الإدارة الذاتية (ASN) المرتبطة بالمؤسسات، مما يساعد في تحديد النطاقات الواسعة لبروتوكولات الإنترنت (IP Ranges) للهدف بشكل دقيق. go install github.com/projectdiscovery/asnmap/cmd/asnmap@latest
shodan Internet scanner محرك بحث مخصص للأجهزة المتصلة بالإنترنت (IoT، خوادم، أنظمة تحكم). واجهة API الخاصة به توفر كنزاً من المعلومات عن المنافذ والتقنيات المكشوفة. pip install shodan
censys Internet scanner محرك بحث مشابه لـ Shodan يتيح لك الاستعلام عن بيانات الخوادم وشهادات SSL لاكتشاف البنية التحتية والموارد المرتبطة بالهدف عبر الإنترنت. pip install censys

10.7 Post-Exploitation & Privilege Escalation Tools

بمجرد حصولك على وصول مبدئي (Initial Access)، تُستخدم هذه الأدوات لتثبيت التواجد، تصعيد الصلاحيات لتصبح Administrator/Root، والتحرك الجانبي داخل الشبكة.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
linpeas Linux privesc enumeration أداة نصية (Script) شاملة لاكتشاف ثغرات ومسارات تصعيد الصلاحيات (Privilege Escalation) على أنظمة Linux، توفر تقريراً ملوناً ومفصلاً يُبرز النقاط الحرجة. curl -L https://github.com/peass-ng/PEASS-ng/releases/latest/download/linpeas.sh
winpeas Windows privesc enumeration النسخة الخاصة بأنظمة Windows من عائلة PEASS، تستعرض إعدادات النظام للبحث عن مسارات لتصعيد الصلاحيات محلياً واكتشاف التكوينات الخاطئة. Download from PEASS-ng releases
linenum Linux enumeration أداة قديمة وموثوقة لجمع المعلومات من خوادم Linux والبحث عن التكوينات الخاطئة (Misconfigurations) والأذونات الضعيفة لتسهيل تصعيد الصلاحيات. git clone https://github.com/rebootuser/LinEnum.git
linux-exploit-suggester Kernel exploit suggestions أداة تقوم بتحليل إصدار النواة (Kernel) لنظام Linux وتقترح قائمة بالثغرات العامة (CVEs) القابلة للاستغلال لتصعيد الصلاحيات محلياً. git clone https://github.com/mzet-/linux-exploit-suggester.git
seatbelt Windows enumeration أداة من تطوير GhostPack مكتوبة بـ C#، تقوم بجمع "بيانات الأمان" من أنظمة Windows لاستكشاف التكوينات القابلة للاستغلال دون تنفيذ أي أوامر ضارة. git clone https://github.com/GhostPack/Seatbelt.git
powerup Windows privesc checks وحدة برمجية ضمن PowerSploit تعتمد على PowerShell لاكتشاف واستغلال مسارات تصعيد الصلاحيات الشائعة في بيئة Windows (مثل مسارات الخدمات غير المحمية). Included in PowerSploit
bloodhound AD visualization أداة متقدمة تعتمد على نظرية المخططات (Graph Theory) لتصور وتحليل العلاقات، المجموعات، وصلاحيات المستخدمين في بيئة Active Directory لاكتشاف مسارات السيطرة. apt install bloodhound
sharphound AD data collector أداة تجميع البيانات (Collector) الرسمية لـ BloodHound، تقوم باستخراج بيانات الجلسات، المستخدمين، والمجموعات من الـ AD وتجهيزها للتحليل. Download from BloodHound releases
mimikatz Credential extraction أداة أسطورية لاستخراج كلمات المرور الواضحة (Clear-text)، ورموز التجزئة (Hashes)، و تذاكر Kerberos من ذاكرة النظام (LSASS) في أنظمة Windows. https://github.com/gentilkiwi/mimikatz/releases
netexec Network exploitation أداة قوية (المعروفة سابقاً بـ CrackMapExec) لاستغلال بروتوكولات شبكات Windows مثل SMB و WMI للتحرك الجانبي (Lateral Movement) والتحقق من كلمات المرور. pip install netexec
chisel TCP tunneling أداة قوية جداً لإنشاء أنفاق (Tunnels) متقدمة لنقل البيانات عبر TCP أو UDP، مغلفة ومخفية داخل طلبات HTTP، لتجاوز جدران الحماية المعقدة للشبكات. go install github.com/jpillora/chisel@latest
ligolo-ng Advanced tunneling أداة متطورة وحديثة لإدارة الأنفاق والـ Pivoting (توجيه حركة المرور)، توفر واجهات شبكة وهمية (TUN interfaces) لتسهيل الاستكشاف الداخلي براحة كاملة. go install github.com/nicocha30/ligolo-ng/cmd/ligolo-ng@latest
proxychains Proxy chaining أداة لسطر الأوامر تجبر أي برنامج على توجيه اتصالاته عبر خوادم بروكسي (SOCKS/HTTP) لتسهيل الوصول إلى الشبكات الداخلية المخفية أثناء الـ Pivoting. apt install proxychains4
metasploit Exploitation framework إطار عمل (Framework) متكامل لتطوير وتنفيذ الثغرات (Exploits)، يحتوي على آلاف الحمولات (Payloads) والأدوات المساعدة للاختراق وتصعيد الصلاحيات. apt install metasploit-framework

10.8 Password Cracking & Brute Force

في حال الحصول على Hashes أو الحاجة لتخمين كلمات المرور للوصول إلى الخدمات، تُعتبر هذه الأدوات الخيار الأمثل للمهاجمين ومختبري الاختراق.

الأداة (Tool) الغرض (Purpose) الشرح التقني (Deep Explanation) أمر التثبيت (Installation)
hashcat Password cracking أداة كسر كلمات المرور (Password Cracker) الأسرع والأكثر تقدماً، تعتمد على معالجات كروت الشاشة (GPU) لدعم مئات الخوارزميات، وتعتبر المعيار الصناعي للكسر. apt install hashcat
john (jtr) Password cracking "John the Ripper"، أداة كلاسيكية وشائعة جداً لكسر كلمات المرور، تدعم العديد من التنسيقات وتُستخدم غالباً للكسر المعتمد على المعالج المركزي (CPU). apt install john
hydra Network login brute force أداة سريعة لتخمين بيانات الدخول (Brute-forcer) تدعم عشرات البروتوكولات الشبكية (مثل SSH, FTP, HTTP-Auth) وتقوم بتجربة القواميس (Wordlists) عبر الشبكة. apt install hydra
medusa Parallel login brute force أداة مشابهة لـ Hydra لتخمين كلمات المرور عبر الشبكة، تتميز ببعض التحسينات في الاستقرار والتوازي المتقدم أثناء عمليات الفحص المتعددة. apt install medusa
burp suite intruder Web login brute force أداة مدمجة داخل Burp Suite مخصصة لتنفيذ هجمات التخمين (Brute-force) والهجمات المخصصة على تطبيقات الويب بسهولة وتخصيص عالٍ جداً. See above
ffuf Web fuzzing/brute force على الرغم من أنها أداة Fuzzing بشكل أساسي، إلا أنه يمكن استخدامها كأداة Brute-force سريعة ومرنة لصفحات تسجيل الدخول وتخطي القيود لتطبيقات الويب. See above

10.10 GTFOBins & LOLBAS

مشاريع أساسية تساعد في تجاوز الحماية وتصعيد الصلاحيات (Privilege Escalation) عبر استغلال الملفات التنفيذية الشرعية المتواجدة أصلاً في النظام، مما يُعرف بأسلوب التعايش مع موارد النظام (Living Off The Land).

GTFOBins: Unix binaries that can be exploited for privesc

مستودع شامل لثنائيات Unix/Linux التي تسمح بتخطي القيود وتصعيد الصلاحيات عبر استغلال إعدادات SUID أو قدرات أخرى (Capabilities).

  • URL: https://gtfobins.github.io/
  • Usage: Search for SUID binary name, get exploitation command

LOLBAS: Living Off The Land Binaries (Windows)

مشروع يركز على أدوات Windows الأصلية التي يمكن للمهاجمين استغلالها لتنفيذ تعليمات برمجية، تنزيل ملفات، أو تخطي الحمايات (مثل AMSI, AppLocker).

  • URL: https://lolbas-project.github.io/
  • Usage: Search for Windows binary, get abuse techniques

APPENDIX: Quick Reference Commands

Full Recon Pipeline (One-liner style)

هذا خط أنابيب (Pipeline) متكامل يجمع بين الاستكشاف السلبي والنشط بخطوة واحدة باستخدام أدوات ProjectDiscovery وغيرها. هذا النهج يضمن كفاءة وسرعة فائقة في جمع النطاقات المباشرة، فحص المنافذ، اكتشاف المحتوى المخفي، وفحص الثغرات الآلي.

Full Recon Pipeline 
# Complete subdomain enumeration pipeline
subfinder -d target.com -all -silent | amass enum -passive -d target.com -silent | assetfinder --subs-only target.com | sort -u | shuffledns -d target.com -r resolvers.txt -mode resolve | httpx -status-code -title -tech-detect | tee all_subdomains.txt

# Port scan all live hosts
cat all_subdomains.txt | naabu -p - -o ports.txt

# Content discovery on all web hosts
cat all_subdomains.txt | ffuf -u https://FUZZ/FUZZ2 -w -:FUZZ -w /usr/share/seclists/Discovery/Web-Content/common.txt:FUZZ2 -mc 200,301,302,403

# Vulnerability scan
nuclei -list all_subdomains.txt -severity critical,high -o vulns.txt

OSCP-like Internal Assessment

مسار منهجي متسلسل يحاكي أسلوب امتحان OSCP واختبارات الاختراق للشبكات الداخلية. يبدأ من اكتشاف الأجهزة، مرورا بفحص الخدمات والويب، ثم الاستغلال وتصعيد الصلاحيات، وينتهي بتوثيق الاختراق.
