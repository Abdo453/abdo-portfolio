Theory / النظرية

EN

Level 1: Classic command injection. Output is returned in the response. Use ; whoami or ; cat flag.txt.

AR

المستوى 2: حقن أعمى (Blind). السيرفر لا يرجع مخرجات الأمر المحقون بل يظهر فقط نجاح/فشل الـ ping. استخدم الـ Time Delay مثل ; sleep 5 للاستكشاف.

EN

Level 2: Blind command injection. No command output is returned. Infer command execution using time delays like ; sleep 5.

EN

Level 3: WAF blocks spaces. Bypass space filters using shell variables like $IFS.
