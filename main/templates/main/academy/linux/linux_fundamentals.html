AR

📚 Topics & Commands / المواضيع والأوامر

1. Basic Navigation / التنقل الأساسي

AR

التنقل عبر المجلدات، معرفة المجلد الحالي، وعرض الملفات المخفية:

EN

Moving across directories, displaying working directory paths, and listing hidden files:

pwd                    # Print working directory
cd /var/log            # Change directory to /var/log
ls -la                 # List all files including hidden ones
cd ~                   # Go to home directory
cd ..                  # Go up one level
cd -                   # Toggle back to previous directory

2. File Operations / إدارة الملفات

AR

إنشاء وتعديل ونسخ وحذف الملفات والمجلدات:

EN

Creating, copying, moving, and recursively deleting files or folders:

touch file.txt         # Create empty file
mkdir new_dir          # Create new directory
rm -rf junk_dir        # Force recursive delete of directory
cp file.txt copy.txt   # Copy file
mv file.txt /tmp/      # Move file
cat /etc/hosts         # Output file contents
head -n 5 /etc/passwd  # View first 5 lines
tail -n 5 /etc/passwd  # View last 5 lines
less /var/log/syslog   # View file page by page

3. Permissions / الصلاحيات والملكيات

AR

تعديل صلاحيات القراءة والكتابة والتشغيل للمستخدمين والمجموعات:

EN

Changing read, write, and execute bits for owners, groups, and others:

chmod 777 exploit.sh   # Read, write, execute for all (Dangerous!)
chmod 755 script.py    # Full permissions for owner, read/execute for others
chmod 644 config.txt   # Read/write for owner, read-only for others
chown root:root file   # Change owner and group to root

4. Users & Groups / المستخدمين والمجموعات

whoami                 # Display current username
id                     # Display user and group IDs (UID/GID)
groups                 # List user groups
su - admin             # Switch to another user session
sudo -i                # Elevate shell session to root
cat /etc/passwd        # Check all system users list
cat /etc/shadow        # Check encrypted user password hashes (root only)

5. Process Management / إدارة العمليات

ps aux                 # Display all active system processes
top                    # Interactive process manager (CPU/Memory usage)
kill -9 [PID]          # Force terminate a process by ID
jobs                   # List background job processes
fg %1                  # Bring background job to foreground
bg %1                  # Resume suspended job in background

6. Networking / الشبكات والاتصالات

ip a                   # View network interfaces and IP addresses
ss -antp               # Show TCP listening ports with process IDs
netstat -tulnp         # Alternative way to check open ports
ping -c 3 8.8.8.8      # Check host reachability
curl http://test.com   # Fetch HTTP page contents
wget http://test.com   # Download file via HTTP
nc -lvnp 4444          # Start netcat listener for incoming shells

7. Text Processing / معالجة وفلترة النصوص

grep "admin" log.txt   # Extract lines containing 'admin'
awk -F: '{print $1}'   # Split fields (e.g., extracting usernames from passwd)
sed 's/old/new/g' file # Replace strings inline
cut -d' ' -f2 file.txt # Extract specific column by delimiter
sort | uniq -c         # Sort lines and count unique occurrences
wc -l files.txt        # Count total lines of a file
xargs                  # Build and execute commands from standard input

8. Package Management / إدارة حزم التثبيت

apt update && apt install nmap     # Debian/Ubuntu package managers
dpkg -i tool.deb                   # Install local debian package
yum install nmap                   # RedHat/CentOS package managers
rpm -ivh tool.rpm                  # Install local RPM package
