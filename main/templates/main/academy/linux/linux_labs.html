Theory & Mission / النظرية والمهمة

AR

المستوى 1: استخدم ls و cd و cat لتصفح المجلدات وقراءة ملف العلم الأول flag1.txt.

EN

Level 1: Use ls, cd, and cat to browse directories and read the first flag file flag1.txt.

AR

المستوى 2: هناك ملف باسم secret.txt محمي الصلاحيات في مجلد secrets. استخدم chmod لتعديل صلاحيات قراءته واستخراج العلم الثاني.

EN

Level 2: A file named secret.txt in secrets dir is restricted. Use chmod to alter permissions to read it.

AR

المستوى 3: ابحث عن ملفات SUID بالدالة find / -perm -4000. استغل ملف find لتشغيل shell مرتفع الصلاحية.

EN

Level 3: Search for SUID files using find / -perm -4000. Exploit the find binary to spawn a privileged shell.

AR

المستوى 4: ابحث عن ملفات Cron المجدولة القابلة للكتابة في /var/tmp. اكتب أمراً فيها لينفذ تلقائياً كمسؤول.

EN

Level 4: Look for writable scheduled Cron scripts in /var/tmp. Write commands inside them to execute as admin.

AR

المستوى 5: تفقد صلاحيات sudo بالأمر sudo -l. استغل أداة awk لإنشاء shell للـ root.

EN

Level 5: Check sudo privileges via sudo -l. Exploit the allowed awk binary to spawn a root shell.

AR

المستوى 6: استغلال ثغرة النواة (Kernel Exploit). تفقد إصدار النواة بـ uname -a، ثم ابحث واستغل الثغرة عبر تشغيل ملف ثغرة محاكي ./dirty_cow_exploit.

EN

Level 6: Kernel Exploit. Check the kernel version via uname -a, identify vulnerabilities, and execute the simulated exploit payload ./dirty_cow_exploit to escalate to root.
