Cloud Security — AWS, GCP, Azure

CRITICAL OWASP Cloud Top 10
AR

Cloud Security تختص بدراسة الثغرات والأخطاء الإدارية والتنظيمية في بيئات السحاب (AWS, Azure, GCP). تشمل الثغرات شيوعاً فتح حاويات التخزين (Open S3 Buckets) للعامة، وإعدادات الهوية غير الآمنة (IAM Misconfigurations)، وتسريب رموز الوصول السحابية عبر ثغرات SSRF.

EN

Cloud Security explores vulnerabilities, misconfigurations, and privilege issues in cloud architectures (AWS, Azure, GCP). Common attacks include exposing data via open S3 buckets, overly permissive IAM roles, and leaking cloud access keys via SSRF.

أشهر ثغرات بيئات السحاب / Vulnerabilities

Open S3 Buckets / Storage

ترك حاويات التخزين مفتوحة بدون صلاحيات تمنع القراءة أو الكتابة العامة.

IAM Privilege Escalation

ترقية صلاحيات حساب سحابي بسيط لمدير النظام (Administrator) بسبب صلاحيات IAM فضفاضة.

Cloud Metadata Leakage

قراءة بيانات تعريف السيرفر (Metadata Service) لاستخراج مفاتيح الوصول المؤقتة للـ IAM Role.

Container Escape

الهروب من الحاويات (Docker/K8s) إلى نظام التشغيل الأساسي للمضيف (Host OS).
