IDOR — Insecure Direct Object Reference

HIGH OWASP A01
AR

IDOR يحدث عندما يستطيع المستخدم الوصول لبيانات أو موارد تخص مستخدمين آخرين بتغيير معرّف (ID, UUID, filename) في الطلب — بدون فحص صلاحيات كافٍ من السيرفر.

EN

IDOR happens when a user accesses another user's data by modifying an object identifier (ID, UUID, filename) in the request — without adequate server-side authorization checks.

أنواع IDOR / IDOR Types

URL Parameter IDOR

API / JSON Body

Function-Level BAC
