📖 Lesson 1: Passive vs Active Recon

Passive recon involves gathering information without touching the target's infrastructure (e.g. WHOIS, OSINT, Shodan). Active recon directly interacts with the target (e.g. Port scanning, Directory fuzzing).

📖 Lesson 2: OSINT & DNS Enumeration

Using tools like Amass and Subfinder, we can discover hidden subdomains. Subdomain Takeovers happen when a CNAME points to an unclaimed cloud resource (like an old S3 bucket or GitHub Pages site).

🧪 Lab: Find Subdomains

Try running the following command in your terminal simulator:

subfinder -d hackerone.com
