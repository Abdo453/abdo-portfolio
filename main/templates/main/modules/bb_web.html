Web Pentesting Module

Attacking web applications following OWASP standards.

📖 Lesson 1: SQL Injection (SQLi)

SQL Injection occurs when user input is improperly sanitized before being executed in a database query. It can lead to data exfiltration, authentication bypass, and remote code execution.

📖 Lesson 2: Cross-Site Scripting (XSS)

XSS allows attackers to execute malicious JavaScript in the victim's browser. Always test inputs that are reflected on the page without encoding.

🧪 Lab: XSS Payload Testing

Try injecting a basic XSS payload into a search parameter:

<script>alert(document.domain)</script>
