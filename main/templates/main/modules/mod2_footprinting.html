مقارنة: الاستطلاع النشط مقابل السلبي

🕵️ الاستطلاع السلبي (Passive)
  • جمع المعلومات دون الاتصال المباشر بالهدف (Stealthy).
  • استخدام محركات البحث، مواقع التوظيف، و Archive.org.
  • البحث في السجلات العامة مثل WHOIS.
  • لا يُسجل في الـ Logs الخاصة بالهدف.
🚨 الاستطلاع النشط (Active)
  • يتطلب الاتصال المباشر بخوادم وأنظمة الهدف.
  • محاولات نقل النطاق (DNS Zone Transfer).
  • الهندسة الاجتماعية المباشرة (Social Engineering).
  • خطر تسجيل الأنشطة واكتشاف المهاجم.

الاستخبارات مفتوحة المصدر (OSINT) و Dorking

Google Hacking (Dorking)

استخدام مشغلات بحث متقدمة لاستخراج معلومات أو ملفات غير مفهرسة بشكل صحيح.

Google Search Queries
# البحث عن مسارات الدليل المفتوحة
intitle:"index of" "target.com"

# البحث عن ملفات إعدادات وكلمات مرور
site:target.com ext:txt | ext:log | ext:env | ext:sql

# البحث عن صفحات لوحة التحكم
site:target.com inurl:admin | inurl:login

أدوات الاستطلاع العميقة (Recon Arsenal)

1. أداة Recon-ng

إطار عمل مخصص للـ OSINT مشابه لواجهة Metasploit.

Recon-ng Console
recon-ng
workspaces create target_corp
modules load recon/domains-hosts/hackertarget
options set SOURCE target.com
run
show hosts

2. أداة TheHarvester

أداة جبارة لاستخراج الإيميلات، النطاقات الفرعية، وعناوين الـ IP من مصادر مختلفة.

Bash Shell
theHarvester -d target.com -b google,linkedin,bing -l 500

3. محرك Shodan

محرك بحث أجهزة الـ IoT والخوادم المربوطة بالإنترنت.

Shodan CLI
# تهيئة الأداة
shodan init <API_KEY>

# البحث عن خوادم أباتشي في دبي
shodan search "apache city:Dubai"

# إحصائيات الثغرات للشركة
shodan stats vuln:cve-2014-0160 org:"Target Corp"

4. جمع معلومات DNS (DNS Footprinting)

البحث عن سجلات الـ DNS واستخراج الـ Subdomains المخفية.

Dig & Fierce
# جلب سجلات الميل (MX)
dig mx target.com

# محاولة نقل نطاق الخادم للحصول على نسخة كاملة من القاعدة
dig axfr @ns1.target.com target.com

# الفحص العنيف للنطاقات
fierce --domain target.com

استراتيجيات الحماية والتخفيف (Mitigation)

التهديد الاستطلاعي (Threat) الحل التقني للحماية (Mitigation)
DNS Zone Transfer تعطيل Zone Transfer (AXFR) بحيث لا يُسمح بها إلا لخوادم الـ Secondary DNS.
Directory Listing إغلاق تصفح الدلائل في إعدادات (Apache/Nginx/IIS) وإخفاء الصفحات الداخلية.
Banner Grabbing & Server Info إخفاء إصدارات الخادم عبر تعديل (HTTP Server Headers) واستخدام صفحات أخطاء مخصصة.
Metadata Leakage استخدام أدوات تنظيف البيانات لإزالة البيانات الوصفية من الـ PDFs والصور قبل نشرها (مثل أسماء الموظفين والبرامج).
