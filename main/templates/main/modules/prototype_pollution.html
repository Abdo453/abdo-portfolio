📚 النظرية — مفهوم الـ Prototype Inheritance وتلويث الكائن الأب

تعتمد لغة جافاسكريبت (JavaScript) على نمط الوراثة القائم على النماذج الأولية (Prototype-based Inheritance). ترتبط جميع كائنات جافاسكريبت بكائن أب مشترك يُدعى Object.prototype.

🧪 ثغرة تلوث النموذج الأولي (Prototype Pollution)

تحدث الثغرة عندما يقوم التطبيق بدمج كائنات البيانات (مثل دوال الـ Merge أو Clone أو Path assignment) بناءً على مُدخلات غير مصفاة من المستخدم، مما يتيح له استخدام الكلمة المفتاحية __proto__ أو constructor.prototype للإشارة للكائن الرئيسي الأب وتثبيت قيم جديدة عليه.

🔥 أثر التلوث على بقية الكائنات

بما أن جميع الكائنات ترث من الكائن الأب، فإن أي خاصية يتم تلويثها عليه (مثال: حقن الخاصية "isAdmin": true) ستظهر تلقائياً في جميع الكائنات البرمجية النشطة في التطبيق، مما يؤدي لتخطي القيود الأمنية أو تصعيد الصلاحيات.

🔬 تشريح خطوات الهجوم والوصول لـ RCE في Node.js

لا يقتصر أثر التلوث على تصعيد الصلاحيات فحسب، بل يمكن أن يؤدي إلى **Remote Code Execution (RCE)** عند دمجها مع مكتبات خارجية (مثل محركات القوالب EJS أو Pug) أو عند تلويث المتغيرات البيئية لـ Node.js:

1. Target Vulnerable Code (PHP/Node merge logic): 
// Vulnerable recursive merge function in Node.js
function merge(target, source) {
    for (let key in source) {
        if (typeof target[key] === 'object' && typeof source[key] === 'object') {
            merge(target[key], source[key]);
        } else {
            target[key] = source[key];
        }
    }
}
2. JSON Payload sent to server:
{ "__proto__": { "isAdmin": true, "shell": "node", "NODE_OPTIONS": "--require /tmp/exploit.js" } }

🛡️ الحماية والدفاع — تحصين كائنات جافاسكريبت

✅ استخدام كائنات بدون نموذج أولي (Null Prototypes)

إنشاء الكائنات التي لا ترث من Object.prototype كلياً لتفادي التلوث: 

// Safe object creation with no prototype
const obj = Object.create(null);
obj.name = "test"; // Safe from __proto__ attacks

🔒 تصفية المفاتيح وتجميد الكائنات

  • Keys Whitelisting/Sanitization: فحص وتصفية المفاتيح لمنع معالجة الكلمات المفتاحية الخطرة كـ __proto__ و constructor.
  • Object.freeze: تجميد الكائن الرئيسي لمنع أي تعديل أو إضافة لخصائصه: Object.freeze(Object.prototype).
