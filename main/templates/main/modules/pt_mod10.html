10. الثغرات المتقدمة (Advanced Vulnerabilities)

فهم واستغلال الثغرات المعقدة مثل SSRF, SSTI, XXE, Request Smuggling, و Cache Poisoning.

[Theory] النظرية والمفاهيم

الثغرات المتقدمة في تطبيقات الويب تتجاوز مجرد إدخال نصوص خبيثة (مثل XSS أو SQLi) وتستهدف البنية التحتية، أو طريقة معالجة البيانات المعقدة، أو تباين تفسير بروتوكول HTTP بين الخوادم.

  • SSRF (Server-Side Request Forgery): تحدث عندما يتمكن المهاجم من إجبار الخادم على إرسال طلبات HTTP (أو بروتوكولات أخرى) نيابة عنه، مما يسمح بالوصول إلى شبكات داخلية، أو خدمات محلية (مثل 169.254.169.254 في البيئات السحابية).
  • SSTI (Server-Side Template Injection): تنشأ عندما يتم تمرير مدخلات المستخدم مباشرة إلى محرك القوالب (Template Engine) مثل Jinja2، Twig، أو FreeMarker. يمكن للمهاجم حقن تعبيرات برمجية يتم تنفيذها على الخادم، مما يؤدي غالباً إلى تنفيذ أوامر عن بعد (RCE).
  • XXE (XML External Entity): تظهر عند معالجة بيانات XML تحتوي على كيانات خارجية (External Entities) غير مقيدة. يمكن استغلالها لقراءة ملفات محلية على الخادم (مثل /etc/passwd)، أو تنفيذ SSRF، أو التسبب في حرمان من الخدمة (Billion Laughs Attack).
  • HTTP Request Smuggling: تحدث بسبب اختلاف طريقة تفسير الخوادم الأمامية (Front-end Proxies/Load Balancers) والخوادم الخلفية (Back-end Servers) لترويسات HTTP (تحديداً Content-Length و Transfer-Encoding). مما يتيح للمهاجم "تهريب" طلب مخفي ضمن طلب شرعي للوصول إلى بيانات مستخدمين آخرين أو تجاوز القيود الأمنية.
  • Web Cache Poisoning: استغلال كيفية تخزين الخوادم للنسخ المخبأة (Caching) من خلال حقن محتوى خبيث (عبر مدخلات غير مفهرسة Unkeyed Inputs) يتم حفظه وعرضه للمستخدمين الآخرين.

[Attack Flow] مسار الهجوم

رسم توضيحي لعملية هجوم SSRF للوصول إلى معلومات حساسة من خادم داخلي أو سحابي:

    deactivate WebServer

[Manual Testing] الاختبار اليدوي

يعتمد الاختبار اليدوي على مراقبة سلوك التطبيق والبحث عن استجابات غير متوقعة أو تأخيرات زمنية.

  • لاختبار SSRF: ابحث عن أي مدخلات تقبل روابط (URL) أو تعالج ملفات خارجية. جرب إدخال روابط تشير إلى الخادم المحلي (localhost, 127.0.0.1) أو خدمات معروفة على الشبكة المحلية، وراقب هل التطبيق يحاول جلب المحتوى. يمكنك استخدام Burp Collaborator لمراقبة التفاعلات غير المباشرة (Out-of-Band).
  • لاختبار SSTI: ابحث عن المدخلات التي تنعكس في صفحة الويب وجرب إدخال عمليات حسابية برمجية مخصصة لمحركات القوالب، مثل {{7*7}} أو ${7*7} أو <%= 7*7 %>. إذا تم إرجاع النتيجة 49، فهذا دليل قوي على وجود الثغرة.
  • لاختبار XXE: إذا كان التطبيق يعالج بيانات XML، قم باعتراض الطلب وحقن تعريف DOCTYPE يحتوي على كيان خارجي (ENTITY) يشير إلى ملف محلي (مثل file:///etc/passwd).
  • لاختبار Request Smuggling: يتطلب الأمر إرسال طلبات تحتوي على ترويستي Content-Length و Transfer-Encoding: chunked (تقنيات CL.TE أو TE.CL) ومراقبة تأخيرات الاستجابة أو الأخطاء التي تدل على انقطاع المزامنة بين الخوادم.

[Automation] الأتمتة والأدوات

استخدام الأدوات المتخصصة يمكن أن يسرع من عملية اكتشاف وتأكيد هذه الثغرات المعقدة.

  • Burp Suite Professional: يحتوي على إضافات قوية مثل HTTP Request Smuggler و Param Miner (لاكتشاف مدخلات Cache Poisoning).
  • Tplmap: أداة متخصصة في اكتشاف واستغلال ثغرات SSTI بناءً على محركات القوالب المختلفة.
  • XXEInjector: أداة لأتمتة حقن كيانات XML بطرق مختلفة واستخراج البيانات خارج النطاق (OOB).
  • SSRFmap: أداة لاستغلال ثغرات SSRF لتنفيذ أوامر أو مسح الشبكة الداخلية.

[Tools] الأوامر والأدوات التطبيقية

أمثلة على أوامر استخدام بعض الأدوات المتقدمة (للأغراض التعليمية المصرح بها فقط):

Tplmap - SSTI Scanner
python3 tplmap.py -u "http://target.com/page?name=user" --os-shell
SSRFmap - SSRF Exploitation
python3 ssrfmap.py -r request.txt -p url -m portscan
Param Miner (Burp Suite Extension)
يستخدم كإضافة داخل Burp Suite عبر واجهة المستخدم (GUI) للبحث التلقائي عن المعلمات المخفية (Unkeyed Parameters) التي قد تؤدي إلى Cache Poisoning.

[Lab] بيئة التدريب والتطبيق (Labs)

أفضل المنصات للتدرب على هذه الثغرات بشكل عملي وآمن:

  • PortSwigger Web Security Academy: يوفر معامل ممتازة ومجانية وتفصيلية لجميع هذه الثغرات المتقدمة. تعتبر المرجع الأول عالمياً في هذا المجال.
  • HackTheBox: يحتوي على العديد من الآلات وتحديات الويب التي تتطلب دمج هذه الثغرات المتقدمة للوصول إلى الصلاحيات الكاملة.
  • TryHackMe: يحتوي على غرف تعليمية تطبيقية مثل "SSRF", و "SSTI" مخصصة للمبتدئين والمتوسطين.

[Quiz] اختبر معلوماتك
