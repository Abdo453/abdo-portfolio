① Passive Collection
# Passive subdomain discovery
subfinder -d target.com -all -recursive -silent -o subs.txt
amass enum -passive -d target.com -o amass.txt
assetfinder --subs-only target.com > assetfinder.txt

# Merge & deduplicate all sources
cat subs.txt amass.txt assetfinder.txt | sort -u > all_subs.txt

⚠️ COMMON MISTAKES VS BEST PRACTICES
■ Common Mistakes
✗ الاكتفاء بـ subfinder وحدها فقط
✗ تجاهل 404 responses بدون فحص CNAME
✗ عدم فحص المنافذ غير المعيارية
✗ بدء Active Recon قبل انتهاء Passive
✗ نسيان wildcard: *.target.com
■ Best Practices
✓ دمج 4+ مصادر + sort -u دائماً
✓ فحص CNAME لكل 404 بـ dig
✓ naabu على كل host للمنافذ غير المعيارية
✓ الأولوية الأولى دائماً = dev / staging / admin
✓ gau + waybackurls للأرشيف التاريخي
✓ لاحقاً دائماً Active ← أولاً Passive
