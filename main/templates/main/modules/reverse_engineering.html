📚 النظرية — بنية الملفات التنفيذية وأساسيات الذاكرة والـ Assembly

الهندسة العكسية هي عملية تحليل نظام لفهم طريقة عمله ومكوناته لكتابة كود مكافئ أو كشف الثغرات. في هندسة البرمجيات المغلقة، نتعامل مع لغة الآلة المترجمة (Machine Code) من خلال تمثيلها بلغة التجميع (Assembly).

📁 هياكل الملفات التنفيذية (Binary Formats)

  • PE (Portable Executable): التنسيق الخاص بأنظمة Windows (يحتوي على ترويسات PE، أقسام مثل .text للكود، .data للمتغيرات، وجدول الاستيراد IAT).
  • ELF (Executable and Linkable Format): التنسيق الخاص بأنظمة Linux (يحتوي على أقسام مماثلة وجدول رموز ديناميكي للتواصل مع مكتبات libc).

🧠 المسجلات وإدارة الذاكرة (x86/x64 Registers)

المسجلات (Registers) هي مواقع تخزين فائقة السرعة داخل المعالج (CPU):

  • RAX/EAX: مسجل التراكم الرئيسي، يُستخدم لتخزين قيم الإرجاع للدوال.
  • RSP/ESP: Stack Pointer، يشير دائماً إلى أعلى المكدس (Top of Stack).
  • RBP/EBP: Base Pointer، يشير إلى قاعدة الإطار الحالي للدالة في المكدس.
  • RIP/EIP: Instruction Pointer، أهم مسجل؛ يحتوي على عنوان الأمر التالي المراد تنفيذه.

🔬 تشريح خطوات التحليل — من الملف الثنائي إلى الكود المصدرى

1️⃣
Static Analysis & File Identification

فحص الملف دون تشغيله للتعرف على نوعه وهيكله والمكتبات المستوردة واستخراج النصوص الثابتة (Strings).

# Identify file type and architecture file target_bin # Extract readable ASCII strings (looking for passwords, URLs, keys) strings target_bin | grep -i "pass" # Check binary protections (NX, Canary, ASLR, PIE) using checksec checksec --file=target_bin
2️⃣
Decompilation (Ghidra / IDA)

تغذية الملف لأداة الهندسة العكسية لتوليد كود تقريبي بلغة C (Decompiled Code) يسهل فهمه وتتبع منطق عمل التطبيق.

  • تحديد دالة البداية main أو دوال التحقق من التراخيص.
  • مراقبة مقارنات نصوص كلمات المرور (مثل strcmp).
3️⃣
Dynamic Analysis & Debugging (GDB)

تشغيل الملف تحت مصحح أخطاء ومراقبة قيم الذاكرة والمسجلات في الوقت الفعلي أثناء وضع نقاط التوقف (Breakpoints).

# Start debugging with GDB (using peda/pwndbg extension) gdb ./target_bin # Set breakpoint at main function break main # Run the program run # Step into instructions stepi # View registers state info registers
