APPENDIX: Quick Reference Commands
Full Recon Pipeline (One-liner style)
# Complete subdomain enumeration pipelinesubfinder -d target.com -all -silent | amass enum -passive -d target.com -silent | assetfinder --subs-only target.com | sort -u | shuffledns -d target.com -r resolvers.txt -mode resolve | httpx -status-code -title -tech-detect | tee all_subdomains.txt# Port scan all live hostscat all_subdomains.txt | naabu -p - -o ports.txt# Content discovery on all web hostscat all_subdomains.txt | ffuf -u https://FUZZ/FUZZ2 -w -:FUZZ -w /usr/share/seclists/Discovery/Web-Content/common.txt:FUZZ2 -mc 200,301,302,403# Vulnerability scannuclei -list all_subdomains.txt -severity critical,high -o vulns.txt
OSCP-like Internal Assessment
# 1. Network discoverynmap -sn 192.168.1.0/24 -oG hosts.txt# 2. Port scan live hostsnmap -sC -sV -p- -iL live_hosts.txt -oN full_scan.txt# 3. Service enumeration# (Use NSE scripts based on services found)# 4. Web enumeration# (Use ffuf, gobuster, nikto on web services)# 5. Exploitation# (Use Metasploit, manual exploitation)# 6. Post-exploitation# (Upload LinPEAS/WinPEAS, run them)# (Use BloodHound for AD environments)# 7. Reporting# (Document everything with screenshots)
