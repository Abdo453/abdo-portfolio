2. Content Discovery
2.1 Directory & File Brute Forcing
Feroxbuster (Rust-based, fast):
feroxbuster -u https://target.com/ -w /usr/share/wordlists/seclists/Discovery/Web-Content/raft-small-words.txt -C 404
FFuF (Go-based, highly flexible):
# Directory fuzzingffuf -u https://target.com/FUZZ -w /usr/share/wordlists/seclists/Discovery/Web-Content/common.txt -mc 200,301,302,403# File extension fuzzingffuf -u https://target.com/FUZZ.php -w /usr/share/wordlists/seclists/Discovery/Web-Content/raft-small-words.txt -mc 200# Virtual host fuzzingffuf -u http://target.com -H "Host: FUZZ.target.com" -w subdomains.txt -mc 200
Gobuster Directory Mode:
gobuster dir -u https://target.com -w /path/to/wordlist.txt -x php,html,js,txt -t 50
Dirsearch:
dirsearch -u https://target.com -e php,html,js -w wordlist.txt
2.2 Archive & Historical Analysis
Waybackurls:
echo target.com | waybackurls > old_endpoints.txt
Gau (GetAllUrls):
echo target.com | gau > urls.txt
Katana (Modern crawler):
katana -u https://target.com -jc -jsl -f qurl -o endpoints.txt
2.3 JavaScript Analysis
Key Techniques: - Extract endpoints from JS files - Find API keys, secrets, and credentials - Discover hidden parameters and routes
Tools:
# Using Katana for JS endpoint extractionkatana -u https://target.com -jc -jsl -f url -silent -o js_endpoints.txt# Using xnLinkFinderpython3 xnLinkFinder.py -i target.com -o output.txt# Using jsluicejsluice https://target.com/app.js# Manual grep patternsgrep -r "api\." *.js | grep -oP 'https?://[^"'\`]+'grep -r "fetch\|axios\|XMLHttpRequest" *.js
2.4 Parameter Discovery
Arjun:
python3 arjun.py -u https://target.com/page.php
ParamSpider:
paramspider --domain target.com
FFuF Parameter Fuzzing:
ffuf -u https://target.com/page.php?FUZZ=test -w /usr/share/wordlists/seclists/Discovery/Web-Content/burp-parameter-names.txt -mc 200
X8 (Hidden parameter discovery):
x8 -u https://target.com/page.php -w parameters.txt
2.5 Content Discovery Pipeline
# Step 1: Crawl and collect URLskatana -u https://target.com -jc -jsl -f url -silent -o all_urls.txt# Step 2: Add historical URLsecho target.com | gau >> all_urls.txt# Step 3: Extract JS endpointscat all_urls.txt | grep "\.js$" | xargs -I {} katana -u {} -jc -jsl -f url >> js_endpoints.txt# Step 4: Directory brute forceffuf -u https://target.com/FUZZ -w /usr/share/wordlists/seclists/Discovery/Web-Content/raft-medium-directories.txt -mc 200,301,302,403 -o directories.txt# Step 5: Parameter discoverycat all_urls.txt | grep "?" | xargs -I {} arjun -u {} -oJ params.json
