5.3 API Authentication Testing
OAuth 2.0 Testing: - Test for missing PKCE in mobile apps - Authorization code interception - Token scope manipulation - Refresh token security
JWT Testing:
# Decode JWTjwt_tool.py <token># Test for algorithm confusionjwt_tool.py -t <token> -X k# Test for weak signing keysjwt_tool.py -t <token> -C -d secrets.txt
API Key Testing: - Key in URL parameters (should be in headers) - Missing key rotation - Keys committed to repositories
5.4 Authorization Testing (BOLA/BFLA)
Broken Object-Level Authorization (BOLA/IDOR):
# Test by changing object IDscurl -H "Authorization: Bearer <token>" https://api.target.com/users/123/orderscurl -H "Authorization: Bearer <token>" https://api.target.com/users/124/orders  # Should fail
Authorization Matrix Testing: - Create matrix of endpoints vs user roles - Test each endpoint with each role - Document expected vs actual responses
